# 叙事节拍库

【调用声明】本库由 Stage 1（编剧）与 Stage 2（分镜）共同调用。
Stage 1 调用：四幕结构、因果连接、软冲突、第三角色四条法则。
Stage 2 调用：节拍→镜头语言映射表（文末）。

---

#### 法则：四幕情绪节拍分布规律

治愈系微观奇观短片遵循「惊奇登场→奇观积累→孤独高潮→释然收束」四幕结构，各幕时长比例与情绪节拍如下：

- **第一幕（世界建立，约15-20%）**：典型节拍为「短暂惊奇→身体笨拙复苏→对未知的试探」。触发条件：角色第一次主动接触未知物，或突发性体积/空间入侵打破静止，角色产生僵直或重心后倾的物理排斥。作用是把角色放进「可触摸的陌生世界」，让观众先确认角色状态再进入探索。
- **第二幕（冲突积累，约20-35%）**：典型节拍为「笨拙误触→感官受挫→缓冲恢复→重整出发」。触发条件：角色主动触碰新奇材质（粉尘喷射/弹性反冲）引发无害物理微冲突，随后通过极度轻柔的物理接触宣告防备卸下。双角色场景中张力通过「始作俑者失误→物理量传导至搭档→搭档被迫响应」堆积。
- **第三幕（情绪爆发，约30-40%）**：典型节拍为「欢脱追逐→边缘悬念→失控与解围」。触发条件：防备卸下后动能转化为奔跑，随后个体脱离群体走向环境边界（水边/高处/黑暗入口），因环境材质改变（湿滑/低摩擦/遮挡）导致重心过度偏移，触发坠落或极限拉扯。
- **第四幕（余韵落幕，约15-25%）**：典型节拍为「奇观狂欢→治愈静谧→俏皮彩蛋」。触发条件：危机解除后残余势能转化为高频弹跳或仪式性动作（碰杯/荡秋千），随后物理位移频率逐渐降低，直至被环境（浓雾/黑夜/黑屏）完全吞噬，动能归零。

节拍切换几乎不靠对白，而靠明确的**物理事件触发**，尤其是：
1. 角色第一次主动接触未知物 → 触发「惊奇→试探」
2. 环境尺度突然放大或外力介入 → 触发「试探→敬畏/压迫」
3. 路径被地形/巨物/遮挡彻底切断 → 触发「压迫→孤独高潮/停滞」
4. 场景或光线的强制重置 → 触发「高潮→收束」

**Stage 1编剧应用示例：**
场景卡「四幕位置」列必须标注当前幕次，「情绪节拍」列从上述节拍词库中选取，「核心物理事件」列的结尾必须为下一幕提供可量化的物理遗留动势。
写法示例：「第一幕转第二幕：A因受惊向后倾倒的惯性尚未停止→B缓慢伸出软体触手停在A面前1厘米处→A的后撤动能被B的静止阻断，转化为试探性的嗅闻」
而非「A一开始很害怕，后来发现B很友好，两人开始一起走」。

#### 法则：场景间因果连接模式

场景与场景之间最稳定的连接器，不是事件本身，而是「事件留下来的物理状态（遗留动势）」。场景切换绝不依赖台词逻辑，而是依赖「未释放完毕的物理动能」作为跨场景连接器。

最常作为场景连接器的物理事件类型：
1. **视线/朝向遗留型**：角色在上一镜头中的仰望、侧望、探头、背对，把「未揭晓之物」带到下一场景，下一场景必须以该视线的「揭晓」或「反打」作为入场。
2. **路径封死型**：悬崖、水边、树干、巨物边缘等直接终止移动，遗留的是「静止+无路可走」的状态，而不是情绪总结。
3. **外力撞击型**：水滴、落物、气流、群体移动等把物理量直接打到角色身上，下一场景承接的不是「被吓到」，而是「尚未完全回弹的形变/余震」。
4. **遮挡揭露型**：大型植物、岩体、木根先遮住信息，再通过角色探头或镜头切换揭晓，遗留的是「部分可见、部分未知」的半揭露状态。

遗留动势必须以**物理量而非情绪词**传递，常见格式：
- 视线还停在上方，身体尚未转正
- 脚步已停，重心仍悬在前倾方向
- 受击后的软体回弹尚未完全结束
- 气流/震动/滴落的尾音仍未消失

**Stage 1编剧应用示例：**
场景卡「核心物理事件」列末尾直接写：
「角色仰头确认上方巨物→颈部朝向保持不变，视线余势留在高处→下一场景以该视线方向的反打揭晓巨物全貌」
而不是写「角色对巨物充满敬畏」。

#### 法则：软冲突与角色合奏逻辑（极度重要）

这类短片的张力不来自角色对抗角色，而来自「一个角色先做出笨拙动作，环境把这个动作放大，再迫使另一个角色被动响应」。在无反派、无对抗的设定下，叙事张力通过「物理失控的连锁传导」与「重心的逐渐崩溃」逐级堆积：

**始作俑者行为模式：**
- 永远是「第一个碰到未知物的人」，行为由探索欲/好奇心驱动，而非精准计划
- 通过笨拙触碰、踩踏、探头、失衡制造第一下扰动
- 失误后不主动求助，自行笨拙恢复，搭档只能旁观或被动承接

**被迫响应者行为模式：**
- 因外力冲量、位移、遮挡、震动而改变姿态
- 响应是应激性的：缩头、僵住、后退、目送、停步
- 其动作不是意志选择，而是身体被迫调整平衡

**三级软冲突升级公式：**
- 第一级（单体微受挫）：角色主动试探环境，因环境特殊物理反馈（弹性反冲/气体喷射）导致单体面部扭曲或短暂僵直
- 第二级（边界危机感）：角色被环境吸引，单方面打破队伍阵型，走向具有潜在物理危险的边界（深水区/高低落差处）
- 第三级（重力失控与极限拉扯）：角色因探身导致重心超过安全阈值，同时脚下材质失去摩擦力，引发不可逆的滑落，同伴施加极强的反向拉扯力，在悬崖边缘形成「重力向下 vs 肌肉向后」的极限软体形变僵持

**最有效的物理连锁公式：**
「A的笨拙主动接触→产生物理扰动→扰动沿环境传导→B被迫改变轨迹/姿势→B的反应又成为下一轮扰动来源」

**Stage 1编剧应用示例：**
场景卡「核心物理事件」列应写：
「A脱离队伍走向光滑长满青苔的巨石边缘探头看鱼（二级冲突）→A重心前倾超过60度，底盘打滑瞬间坠落→B飞扑咬住A的尾部→两人在边缘形成剧烈的弹簧般拉扯颤动（三级冲突）」
而非「A不小心掉下去了，B赶紧跑过去把他拉上来」。

#### 法则：环境道具作为"第三角色"的驱动规律

巨大的自然物体不是背景，它们承担了「比台词更强的叙事功能」，相当于第三角色。环境物体通过其物理属性替代台词推动剧情，强制改变角色行动轨迹：

**第三角色的工作方式：**
1. **遮挡信息**：巨叶、伞盖、木根先遮住视线，再通过局部揭露制造悬念
2. **压制行动**：巨物的体积、重量、边界、阴影强制角色减速、停步、缩身、绕行
3. **改写轨迹**：湿滑、弹性、重力、空隙、边缘感，把角色原本的直线行动改成弯折、停顿、探头、贴边移动
4. **替代对白**：当环境本身足够大、足够重、足够近时，角色不需要「表达情绪」，只要被它改变行动即可

**环境物理属性强制改变行动轨迹的通用规律：**
- 湿滑/低摩擦（天鹅绒般的青苔石/清透浅滩/带露水的巨叶）：让「直走」变成「试探着挪步」，必然触发重心失控，用于制造危机
- 高弹性（巨型蘑菇/柔软草皮）：让「触碰」变成「回弹与二次震颤」，将下落重力转化为夸张反弹升力，用于情绪释放
- 遮挡/形变（巨大荷叶/垂下软枝）：让「直视」变成「探头/绕到侧面」，通过边缘拉扯与形变制造躲猫猫欢脱互动
- 刺激型（花粉/水滴）：直接作用于角色感官，强制产生生理应激反应（喷嚏/闭眼缩脖），无需任何叙事铺垫
- 路径封死（悬崖/黑暗隧道口）：让「前进」变成「静止凝持」，强制双角色从运动模式切换至静止凝视模式

本质上，环境在不断替角色做决定：角色不是自己选择「停、看、绕、缩」，而是被环境的物理属性逼出来的。

**Stage 1编剧应用示例：**
场景卡「地点」列应写：
「向日葵花盘（花粉密度极高，吸入后强制触发喷嚏应激，导致角色失去平衡向后倒去）」
而非只写「向日葵旁边」。

场景卡「核心物理事件」列应写：
「角色踩上极具弹性的巨型菌盖→重力转化为向上的强反冲力→迫使腹部产生滞后的果冻震颤，并被弹至半空进入狂欢状态」
而非「角色看到蘑菇很高兴，开始在上面开心地跳舞」。

#### 节拍→镜头语言映射表（Stage 2分镜专用）

以下映射关系专供分镜阶段参考，将叙事节拍直接转化为可执行的镜头决策（注意：这是内部参考逻辑，不是输出格式，严禁在输出中生成任何表格）：

- **惊奇登场/初次接触** → 景别：大全景→中景 / 机位：平视或微俯拍 / 运镜：固定或缓慢推进 / 时长：4-6s / 转场：遮挡揭露
- **笨拙触发/误触** → 景别：中景→特写 / 机位：平视或略低 / 运镜：固定，切换靠剪辑 / 时长：2-4s / 转场：动作匹配硬切
- **失控升级/弹跳** → 景别：跟随中景 / 机位：跟随角色运动 / 运镜：跟随+轻微晃动 / 时长：3-5s / 转场：速度反差锚点
- **被迫响应（非主动）** → 景别：近景 / 机位：平视，略偏侧 / 运镜：固定，不跟随 / 时长：2-3s / 转场：视线匹配
- **边界危机/悬崖边** → 景别：大全景 / 机位：广角大远景，强调渺小感 / 运镜：固定，让角色变小 / 时长：4-7s / 转场：路径封死型
- **余波收束/静谧** → 景别：大全景 / 机位：固定，高位俯拍 / 运镜：固定，画面逐渐静止 / 时长：5-8s / 转场：渐暗或遮挡

**遗留动势的镜头表达规则：**
- 视线遗留型 → transition_out 写：「镜头停在角色仰望方向，不跟随角色转身」
- 路径封死型 → transition_out 写：「固定机位，角色停在画面边缘，镜头不动」
- 外力撞击型 → transition_out 写：「切出点卡在形变最大值瞬间，下一镜承接回弹余震」
