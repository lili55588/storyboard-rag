## 零、底层写作规范（必须先读）

> 这三条规范是 Seedance 2.0 官方底层逻辑的直接体现，所有提示词必须遵守，优先级高于其他一切技巧。

---

### 规范一：纯中文写作，禁止中英混杂

Seedance 2.0 对中文原生语义的理解精度远高于英文或中英混合写法。英文单词夹在中文句子里，模型会切换到翻译模式解析，语义准确率大幅下降。

```
❌ 错误：dolly-in推镜，穿黑色oversize卫衣的dancer腾空起跳
✅ 正确：镜头缓慢向前推进，穿黑色宽松卫衣的街舞舞者屈膝蓄力腾空跳起
```

**唯一例外**：风格词（如 4K、60fps、16:9）因为是技术参数，可以保留英文数字格式。

---

### 规范二：权重优先级不可逆——运镜必须放最开头

Seedance 2.0 官方确认的权重优先级排序：

```
运镜/视角 > 动作时序 > 主体定义 > 场景环境 > 光影氛围 > 风格调性 > 细节约束
```

**重要理解**：这个优先级排序有两层含义：

**含义一（冲突时谁赢）**：当不同模块的指令互相矛盾时，高权重模块优先被满足。

**含义二（写作顺序）**：运镜必须放在提示词最开头，这样模型首先锁定镜头逻辑，出片率提升30%以上。运镜放在后面非常容易被忽略。

**写作顺序与权重关系的正确理解**：
```
权重优先级 ≠ 写作顺序要按权重倒着排
动作一定是「主体的动作」，主体必须在动作之前出现，
否则模型找不到动作的发出者，会出现生成错位。

正确写作顺序：运镜（最高权重）→ 主体（谁）→ 动作（做什么）→ 后续细节
```

---

### 规范三：单个生成单元提示词长度控制在 120–280 词

| 长度 | 问题 | 后果 |
|---|---|---|
| 少于120词 | 信息不足，随机性过高 | 模型自由发挥，结果不可控 |
| 120–280词 | 最优范围 | 核心权重集中，模型精准执行 |
| 超过280词 | 信息过多，权重稀释 | 核心指令被边缘信息淹没，效果降级 |

**执行方法**：写完提示词后数一遍字数，超出就删掉信息密度最低的描述，不足就补充动作细节或微表情。

## 一、最核心的底层认知升级：两种运镜的本质区别

这是整本手册最重要的一个概念，搞清楚它，运镜问题迎刃而解。

### 路径型运镜 ← AI 当作轨迹指令执行

描述镜头的**具体移动路径和终点**，AI 需要精确计算镜头轨迹。

```
例：镜头从左侧以45度弧线推进，同时向上摇，最终停在人物右肩特写位置
例：镜头从远景缓慢推至中全景，人物停稳后固定
```

**特点**：占用大量算力；动作越复杂，路径型运镜越容易导致动作变形。

---

### 质感型运镜 ← AI 当作风格标签理解

描述镜头的**整体氛围和质感**，不规定具体轨迹，AI 自主解释执行。

```
例：手持摄影，全程晃动，剧烈呼吸感
例：镜头全程环绕推拉，纪录片写实风格
```

**特点**：几乎不占算力；算力全部释放给动作本身；是成片文件高成功率的核心秘密。

---

### 核心使用规则

| 动作复杂度 | 运镜策略 | 原因 |
|---|---|---|
| 极复杂（激烈打斗/高难度技巧） | 固定镜头 + 质感型运镜 | 路径型占算力，动作变形 |
| 中等（走动/追击/对峙） | 路径型 + 质感型叠加 | 动作不复杂，可分算力给轨迹 |
| 极简（静止/对话/情绪） | 路径型为主 | 算力充足，执行精确路径 |

**黄金组合公式**：
```
路径型运镜 → 只放在 clip 的起点（静止状态）或终点（动作完成后）
质感型运镜 → 贯穿全程，作为整体风格标签叠加
```

## 二、制造动感的核心方法：拆 clip + 专属特写 clip

> 单个 clip 内部强化运镜，动感提升有限。真正的动感来自**景别跳跃**和**专属特写 clip 的插入**。

### 专属特写 clip 是什么

在关键动作发生后，专门切入一个 clip，只拍**该动作最具冲击力的局部细节**：

- 拳头命中瞬间的面部反应
- 腿法横扫时脚踝破开雾气的特写
- 人物稳住后眼神的大特写
- 武器/道具的细节特写

这一个 clip 本身不推进剧情，但它创造了**景别跳跃**，这个跳跃本身就是动感的来源。

---

### 标准动感拆分模板（以10秒打斗为例）

❌ **旧方案**：1个clip 10秒，加运镜
```
单clip内部强化运镜 → 有限提升，动作变形风险大
```

✅ **新方案**：拆成3个clip，插入专属特写
```
Clip 1（4秒）· 定场起手   → 路径型推镜（静止承接）
Clip 2（4秒）· 交手核心   → 固定 + 质感晃动（算力全给动作）
Clip 3（3秒）· 冲击特写   → 仰拍/倾斜大特写 + 上摇（景别跳跃）
```

---

### 四种 clip 类型的运镜策略

#### ① 定场 / 收尾 clip（动作极简）→ 路径型运镜

```
镜头从远景缓慢推近至中全景，人物对峙停稳后镜头固定
```
```
打斗结束，双方重新拉开距离，镜头缓慢向后拉开，两人身影在画面中央收小
```

#### ② 激烈打斗 clip（动作极复杂）→ 固定 + 质感型

```
固定中景，手持轻微晃动，呼吸感贯穿全程，
关键撞击瞬间镜头跟着抖动一下
```

#### ③ 走动 / 追击 clip（中等动作）→ 路径型 + 质感叠加

```
镜头跟随人物背影中近景推进（路径），手持轻微晃动（质感）
```

#### ④ 情绪 / 专属特写 clip（近景大特写）→ 路径型微运镜

```
仰拍，倾斜构图20度，镜头从脚踝特写快速上摇至面部大特写，
前景是对手虚化身影，手持晃动
```

## 三、完整工作流（六步骤）

> ⚠️ 本章新增第五步「合并判断」，这是手册最容易被忽略、也最影响实际生成效果的一步。

---

### 【核心概念】叙事分镜 vs 生成单元

整个工作流围绕两个不同层级的"clip"运作，必须先搞清楚：

| 概念 | 用途 | 判断标准 |
|---|---|---|
| **叙事分镜** | 规划阶段使用，景别/运镜/情绪的最小叙事单位 | 任何一个维度变了就新开一个 |
| **生成单元** | 实际丢给即梦的一条提示词 | 景别+运镜不变可合并多个叙事分镜，总时长≤13秒 |

```
叙事分镜是规划工具，生成单元是执行工具。
多个叙事分镜可以合并成一个生成单元一次提交，
但景别或运镜不同的叙事分镜，必须拆成独立生成单元分次提交。
```

---

### 步骤一：通读剧本，建立固定要素库

所有**不变信息**提前标准化，这是跨 clip 一致性的根基。

```
【固定要素库 - 人物】
女主林溪：26岁亚洲女生，鹅蛋脸，弯眉杏眼，齐肩深棕色锁骨发，
穿燕麦色羊绒开衫，内搭米白色针织衫，下身卡其色阔腿裤。

【固定要素库 - 场景】
秋日老城飘窗：木质飘窗，窗外金黄色银杏叶随风慢慢飘落，
午后阳光透过玻璃窗洒进来，室内暖木色调。
```

需要提取：角色完整外形（年龄/性别/发型/服装/核心特征）、场景完整描述（地点/时间/天气/光影/陈设）、叙事断点标记。

---

### 步骤二：按动作单元 / 情绪单元拆 clip

**本质**：一件事 / 一个完整动作 = 一个 clip，根据叙事节奏决定数量。

| 视频节奏 | 100秒合理拆分 | 单 clip 时长 | 适用场景 |
|---|---|---|---|
| 慢节奏文艺片 | 3–6 个 clip | 10–13秒 | 情感短片、诗意叙事 |
| 中节奏剧情片 | 7–10 个 clip | 8–12秒 | 故事短片、日常剧情 |
| 快节奏动作片 | 12–15 个 clip | 4–8秒 | 打斗、快剪预告 |

**拆分判断标准**：
- 景别变了 → 必须拆
- 运镜变了 → 必须拆
- 场景转了 → 必须拆
- 情绪转了 → 必须拆
- 4–6秒快切本身就是叙事语言，不要强行拉长

⚠️ **铁律**：单 clip 绝对上限 **13秒**，留2秒冗余，永远不卡满15秒。

---

### 步骤三：补镜头语言

剧本只写剧情，不写景别和运镜，需要主动补全：

**景别选择**：
- 开场定场 → 全景 / 远景
- 人物出场 / 全身姿态 → 中全景
- 对话 / 上半身动作 → 中景
- 细节 / 情绪 / 冲击反应 → 近景 / 特写

**运镜选择**（按本章第一节规则）：
- 复杂动作 → 固定 + 质感型晃动
- 简单动作 / 静止 → 路径型推 / 拉 / 摇
- 追击 / 跟随 → 跟拍 + 质感叠加
- 高潮特写 → 倾斜构图 + 仰拍 + 上摇

---

### 步骤四：合并判断——叙事分镜 → 生成单元

这一步决定"哪些叙事分镜合并提交，哪些必须单独跑"。

**判断规则**：
```
景别相同 + 运镜相同 → 可以合并，用「随后」「紧接着」「最终」串联动作
景别改变 OR 运镜改变 → 必须拆成独立生成单元，分次提交
```

**合并示例**（同一运镜下连续动作 → 一次生成）：
```
✅ 可合并：男性连续三记重拳（左直拳→右钩拳→左摆拳），全程固定中景+手持晃动
→ 三个叙事分镜合并为一个生成单元，总时长6秒，一次提交
```

**必须拆分示例**（运镜/景别不同 → 分次生成）：
```
❌ 不可合并：
叙事分镜A：仰拍跟推（御剑破云而出）
叙事分镜B：侧面跟飞（穿越山峰）
叙事分镜C：腰部上摇到面部特写（情绪落点）
→ 三种完全不同的运镜，必须拆成三个独立生成单元，分三次提交，后期剪辑拼接
```

**合并后的时长上限**：多个叙事分镜合并后，生成单元总时长≤13秒，不卡满15秒。

---

### 步骤五：按固定公式生成每个生成单元的提示词

**公式（顺序严格不能乱，基于官方权重优先级）**：

```
【第一位】运镜/视角（最高权重，必须放最开头）
  → 例：固定中景，手持强烈晃动 / 镜头缓慢从远景推至中全景

【第二位】主体外形描述（完整，从头到脚，多角色按主次排序）
  → 例：30岁中国男性，寸头，浓眉方颌，穿黑色棉麻练功服，束白腰带，黑色千层底布鞋

【第三位】大动作（时间顺序分步写，从起点到终点，不只给静态结果）
  → 例：男性右腿猛地抬起踢出旋踢，女性急速左闪，旋踢擦过耳侧

【第四位】微表情 / 微动作细节（消弭AI僵硬感）
  → 例：胸口急促起伏，鼻翼微张，发丝被气流吹乱

【第五位】对话状态处理（见第六章规则）

【第六位】场景 + 环境描述（完整复制固定要素库）
  → 例：深山青石武台，四周苍松环绕，薄雾低伏，清晨冷蓝天光

【第七位】光影氛围
  → 例：逆光轮廓清晰，冷蓝色天光，高对比

【第八位】风格调性
  → 例：写实动作风格，纪录片质感，高对比冷色调

【第九位】质量约束（人物必加，场景按需加）
  → 人物必加：面部五官清晰稳定不变形，眼睛嘴型不扭曲，人体结构正常，比例自然，动作连贯不僵硬
  → 注意：负面约束只加最核心1–2条，不要堆砌，模型对正面描述敏感度远高于负面提示词

【最后】时长 + 比例
  → 例：6秒，16:9
```

⚠️ **不要直接抄剧本原文，要转换为动态过程描述**：
```
❌ 剧本原文：「林溪看书」
✅ 正确转换：「林溪低头看着腿上摊开的旧书，手指轻轻翻了一页，看完后慢慢合上放到身侧，肩膀随呼吸轻轻起伏」
```

---

### 关于 [frame_X] 时序标记的使用边界

`[frame_X]` 是一种在动作描述中加入帧级时间节点的写法，例如 `[t=0.5s]出拳 [t=3.2s]击中`。

**正确使用边界**：
```
单个生成单元内有 3 个以上连续复杂动作 → 可以加 [frame_X] 精确控制时序节点
单个生成单元内只有 1–2 个动作 → 用自然语言时序描述就够，强行加 [frame_X] 反而增加冗余、稀释权重
```

**实际大多数情况不需要用**，自然语言时序（"随后""紧接着""同时"）已经足够让模型理解动作顺序。新手不要强行加标记。

---

### 步骤六：一致性 + 衔接 + 时长 + 权重四项检查

1. **一致性**：同一人物 / 场景，所有 clip 固定描述是否一字不差复制？
2. **衔接**：下一个 clip 起点 = 上一个 clip 终点，不跳步。
3. **时长**：所有生成单元在13秒以内，复杂动作留冗余？
4. **权重顺序**：运镜在最开头，风格调性 / 质量约束 / 时长比例放最后？

---

## 四、跨 clip 一致性：四个维度

> 90% 的多 clip 成片翻车，都源于「跨 clip 信息不一致」。

### 维度一：人物一致性

两种方案，选一种执行到底，不能混用：

**方案 A：纯文字全复制**
同一角色，所有用到它的 clip，开头外形描述一字不差复制粘贴，只改动作和运镜，不动主体描述。

**方案 B：参考图锚定**
每一个涉及该人物的 clip 都挂同一张参考图（`@图片X`），缺任何一个 clip 的参考图，模型会按文字重新生成外形，导致换脸。

⚠️ **绝对不能用"人物同上"简写**——无参考图的纯文字稿下，这是换脸 / 换服装的直接原因。

---

### 维度二：场景光线一致性

同一场景同一时间段，场景描述 + 光线描述完全复制，不能简写。

```
❌ 错误：上一个 clip 写「清晨薄雾浅金色晨光」，下一个简写「清晨阳光」
✅ 正确：每个 clip 完整复制「清晨深山古太极平台，青石板，百年古松，浅金色晨光，薄雾缭绕」
```

---

### 维度三：动作衔接

```
❌ 错误（跳步卡顿）：
Clip1 结尾：女生走到茶铺门口停下
Clip2 开头：女生推开木门走进店内  ← 中间缺了「手握把手」的衔接动作

✅ 正确（丝滑不卡）：
Clip1（终点）：…走到茶铺木门门口，停下脚步，右手扶住木门把手
Clip2（起点）：女生右手握住木门把手，轻轻向内推开，侧身走进店内
```

---

### 维度四：跨场景切换

跨场景切换（室外→室内、白天→夜晚）不需要动作承接，但新 clip 开头必须明确交代新的时间 / 地点，否则模型会沿用上一个 clip 的光线和环境续接。

---

## 五、不同题材的分镜策略

### 1. 慢节奏文艺片 / 情感短片

> ⚠️ 3分钟以内均为短视频，空镜和预热节奏规则仍受第八章约束。

- **拆分**：3–6 个生成单元，长镜头保持情绪连贯
- **运镜**：缓慢推 / 拉 / 摇（路径型为主），少用质感晃动
- **提示词重点**：微表情、微动作、光影氛围，大动作简单带过
- **特写策略**：情绪高潮处插入 1 个面部特写生成单元，缓慢推镜放大情绪

**100秒示例拆分**：

| 生成单元 | 内容 | 时长 |
|---|---|---|
| 1 | 主角出场，环境信息写入场景描述 | 8s |
| 2 | 主角近景细节动作 | 13s |
| 3 | 情绪转折特写 | 12s |
| 4 | 人物关系中景 | 13s |
| 5 | 结尾定格 | 8s |
| **合计** | | **54s** |

偏短时：找叙事上值得延展的生成单元重新生成更长版本，或插入 1–2 个细节特写生成单元补足。

---

### 2. 快节奏动作片 / 打斗短片

- **拆分**：12–15 个生成单元，一个小动作一个生成单元
- **运镜**：激烈打斗段用固定 + 质感晃动；定场 / 收尾用路径型推拉摇
- **提示词重点**：动作按时间顺序分步写，每个生成单元只放一个完整小动作，主体描述全复制
- **动感核心**：每个打斗高潮后必插一个专属特写生成单元（反应镜头），景别跳跃制造节奏冲击

**100秒示例拆分**：

| 生成单元 | 内容 | 时长 | 运镜类型 |
|---|---|---|---|
| 1 | 双方出场即对峙，环境写入场景描述 | 4s | 路径型推 + 质感轻晃 |
| 2–4 | 第一回合（3个生成单元） | 4+4+5s | 固定 + 质感晃动 |
| 5 | 反应特写 | 3s | 仰拍倾斜 + 上摇 |
| 6–8 | 第二回合（3个生成单元） | 5+5+5s | 固定 + 质感晃动 |
| 9 | 反应特写 | 3s | 仰拍倾斜 + 上摇 |
| 10 | 高潮击中 | 5s | 固定 + 撞击抖动 |
| 11 | 收尾定格 | 3s | 路径型缓拉 |
| **合计** | | **~46s** | |

### 3. 中节奏故事剧情片

- **拆分**：7–10 个 clip，按叙事节点切割
- **运镜**：对话用固定中景，转场用摇镜，情绪升级用推镜
- **核心注意**：每个情节转折必须拆 clip，不能把转场塞进原情节 clip
- **特写策略**：关键情绪转折点后插入一个近景特写 clip，缓慢推镜

---

### 4. 营销广告短片

- **拆分**：第一个 clip 4–6秒直接放核心钩子（黄金3秒法则），每个卖点一个 clip
- **运镜**：产品特写用推镜，使用场景用跟镜，结尾定格用固定
- **一致性铁律**：产品外形描述和品牌文字在所有 clip 完全一致，产品外观零偏差

## 六、对话 / 台词处理：两种情况

### 情况 A：无音频参考（纯画面视频）

不写台词文字，只描述说话的动作状态：

```
开口说话：「嘴唇轻轻张开，跟着说话节奏微微动着，转头看向对方」
说完停顿：「说完后嘴唇轻轻闭合，等待对方回应」
情绪说话：「语气激动，说话时肩膀微微晃动，眉头轻皱，嘴唇快速动着」
```

### 情况 B：有音频参考（`@音频X` 指定音色）

可以直接写台词原文，配合音色参考实现语音合成：

```
姐姐微笑了一下，低声说：活着就要笑啊……活着已经很难了。
```

⚠️ **没有 `@音频` 的情况下写台词原文，模型会忽略或产生错误口型**。

## 八、短视频节奏铁律（3分钟以内全部适用）

> 3分钟以内均为短视频，以下铁律全部适用。违反任何一条都会直接流失观众。

---

### 铁律一：第一帧必须有人，第一秒必须有动作

短视频没有预热预算。观众在前3秒决定要不要继续看，给环境1秒已经是极限。

```
❌ 错误：空镜8秒定场 → 双人缓步走入 → 停住对视 → 才开始动作
✅ 正确：第一帧两人已在场，第一秒就在踏步试探或直接交手
```

执行方法：把环境信息完全塞进场景描述字段，不用单独开空镜 clip 交代环境。

---

### 铁律二：空镜时长上限 1–2 秒

短视频（50秒以内）中，独立空镜 clip 最多1–2秒，超过就是在赶观众。

```
❌ 几十秒短视频里来一个8秒空镜定场 → 观众当成静止画面直接划走
✅ 必须要空镜时：最多1秒，且要有明显动态元素（风吹雾动、光线变化）
```

更好的方案：完全不单独开空镜 clip，把"薄雾流动 / 晨光透入 / 苍松随风"写进第一个有人 clip 的场景描述里，环境感一样有，时间成本为零。

---

### 铁律三：试探性动作直接删掉

"试探"在现实武打中存在，在短视频提示词里是浪费。一个试探性刺拳+格挡单独占一个5秒 clip，对观众而言等于5秒空白。

```
❌ Clip 2（5秒）：男性探出一记试探性刺拳，女性侧头闪开，两人重新站定审视对方
✅ Clip 2（3秒）：男性率先爆发，右腿猛踢女性腹部，女性硬格挡，冲击力逼退一步
```

第一次交手就全力，压强决定节奏感。

---

### 铁律四：收尾最多 3 秒

动作短视频的收尾只需要一个结果：胜负已定，人物离开或定格。

```
❌ 收尾（5秒）：女性缓缓起身，整理马尾，低头最后看了一眼，转身缓步离开，薄雾散开
✅ 收尾（3秒）：女性起身，俯视倒地男性，转身出画，镜头缓拉固定
```

5秒收尾是文艺片语法，动作短视频用文艺片节奏收尾是前功尽弃。

## 九、时长弹性原则

100秒是目标，允许 ±10秒弹性空间（90–110秒均为合理成片）。

- **偏短（差10–15秒）**：找叙事上值得延展的 clip 重新生成更长版本，或插入1–2个细节 / B-roll / 专属特写 clip
- **偏长（超出10秒以上）**：找信息密度最低的过渡性 clip 剪短或直接去掉
- **不要用淡入淡出凑时长**：最多贡献0.5–1秒，填不了实质性时间缺口

## 十、终版完整检查清单

### 单生成单元层面

- ✅ 提示词是否纯中文？有无中英混杂？（零章规范一）
- ✅ 提示词字数是否在 120–280 词之间？（零章规范三）
- ✅ **运镜/视角在最开头**，风格调性 / 质量约束 / 时长比例在最后？（零章规范二）
- ✅ 主体外形描述完整（年龄 / 五官 / 服装 / 特征到脚），放在运镜之后、动作之前？
- ✅ 动作按时间顺序分步写，从起点到终点，不只给静态结果？
- ✅ 已加微表情 / 生理细节（呼吸 / 眨眼 / 发丝晃动），消弭 AI 僵硬感？
- ✅ 运镜类型正确：复杂动作用固定 + 质感型；简单 / 静止用路径型？
- ✅ 路径型运镜只放在起点（静止状态）或终点（动作完成后），复杂动作过程中不出现路径型？
- ✅ 质感型运镜作为风格标签贯穿全程？
- ✅ 运镜方向与主体动作方向不冲突？
- ✅ 时长合理，不超过13秒，复杂动作留冗余？
- ✅ 人物生成单元已加质量约束（面部五官清晰稳定不变形，人体结构正常，比例自然）？
- ✅ 对话处理正确：有音频→可写台词；无音频→只写嘴部动作状态？

### 合并判断层面（新增）

- ✅ 同一景别 + 同一运镜的叙事分镜是否已合并为一个生成单元？
- ✅ 景别或运镜不同的叙事分镜是否已拆成独立生成单元？
- ✅ 合并后的生成单元总时长是否在13秒以内？

### 跨 clip 层面

- ✅ 一致性方案选定（纯文字全复制 OR 参考图锚定），全程不混用
- ✅ 同一人物所有外形描述，所有 clip 完全一致（全复制，不写"同上"）
- ✅ 有参考图时，涉及该人物的每个 clip 都挂同一张参考图
- ✅ 同一场景光线 / 环境描述，所有 clip 完全一致
- ✅ 上下 clip 动作衔接：下一个起点 = 上一个终点，不跳步
- ✅ 跨场景切换时，新 clip 开头明确交代新时间 / 地点
- ✅ 服装 / 配饰 / 颜色，所有 clip 无漏写或改描述
- ✅ 所有 clip 风格描述和色调一致
- ✅ 高潮打斗 / 情绪爆发后有专属特写 clip 跟进（动感关键）

### 动感检查

- ✅ 打斗段有无插入专属特写 clip？
- ✅ 专属特写 clip 有无使用仰拍 + 倾斜构图 + 上摇组合？
- ✅ 激烈动作 clip 有无使用撞击抖动描述？
- ✅ 定场 / 收尾 clip 有无路径型运镜形成包裹感？

### 短视频节奏检查（新增）

- ✅ 第一帧是否有人？第一秒是否有动作？
- ✅ 有无独立空镜 clip 超过2秒？

- ✅ 对峙 / 定场 clip 是否控制在4秒以内且有持续动态？

## 核心心法总结

**写作规范**：纯中文写作，禁止中英混杂；单个生成单元 120–280 词，不多不少；运镜必须放最开头锁定权重。

**公式顺序**：运镜 → 主体 → 动作 → 微细节 → 对话状态 → 场景 → 光影 → 风格 → 质量约束 → 时长比例。顺序不能乱。

**质量约束**：人物生成必加「面部五官清晰稳定不变形，眼睛嘴型不扭曲，人体结构正常，比例自然，动作连贯不僵硬」。负面约束只加最核心1–2条，不要堆砌。

**一致性**：固定要素库（文字全复制 or 参考图全挂）是防止换脸 / 换服装的根本。运镜 / 主体 / 场景 / 光线全部完整复制，绝对不写"同上"。

**合并判断**：景别+运镜不变 → 多个叙事分镜合并一次提交；景别或运镜改变 → 必须拆成独立生成单元分次提交，后期剪辑拼接。合并上限13秒。

**动感**：不靠单生成单元内部运镜堆叠，靠拆分 + 景别跳跃 + 专属特写生成单元的节奏组合。打斗高潮后必插专属特写生成单元。

**运镜**：路径型管轨迹，质感型管氛围。复杂动作过程中只用质感型保住算力，路径型只放起点和终点。

**节奏**：3分钟以内全是短视频。第一帧有人，第一秒有动作，空镜不超过2秒，试探性动作直接删，收尾不超过3秒。快准狠，每秒都在动。

**台词**：有音频挂音频写台词，没有音频只写嘴部动作状态，不写具体文字。
