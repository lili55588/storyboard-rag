# KB-04 · 固定要素库模板

> **加载阶段**：阶段二（视觉开发）。
> **用途**：为剧本中每个角色和场景生成标准化的"角色卡 + 场景卡"，作为阶段三分镜提示词的一致性锚点。
> **核心原则**：**角色原图优先，文字锁定兜底**。已有角色原图时，角色卡必须极简，只登记 @图片编号、短识别、防跑偏禁令和必要状态；不要重新长篇描写角色外貌。常规生成单元只使用角色名 + @图片编号；只有无角色原图、首次建模或跑偏修正时使用短锚定/完整锁定描述。禁止写"同上"。

---

## 1. 角色卡标准模板

### 1-0 已有角色原图时：极简角色卡（默认）

如果用户已经提供或项目已经登记角色原图，阶段二不要重新写大段外貌。角色一致性交给 @图片。

```markdown
### 角色卡 · [角色名]

**角色原图引用写法（Stage 3 常规使用）**：[角色名]（@图片X）

**短识别兜底（仅无图/跑偏修正时使用，20-40字）**：
[只保留最核心识别点，不扩写身体结构，不发明未设定部位]

**永久附属物**：[如：小圆盾、小剑。没有则写“无”]

**允许变化的状态**：
| 状态 | 触发场景 | 允许变化 |
|---|---|---|
| 默认 | ... | ... |

**禁止跑偏**：[3-6条短禁令，只写真正高风险项]

**参考图登记**：
- @图片X：[角色名]基础原图（最高优先级）
```

极简角色卡要求：

- 不写“完整锁定描述”大段白描，除非明确没有角色原图。
- 不新增角色原设没有的身体部位、左右手、手指、膝盖、尾巴、翅膀、衣物或复杂表情。
- 不生成大量状态图建议；状态图留到阶段三需要首尾帧时用 @图片50+ 补。
- 不把角色卡写成角色圣经；篇幅留给场景卡、道具和参考图编号。

### 模板结构

以下完整模板只在没有角色原图、首次建立角色资产或跑偏修正时使用。

```markdown
### 角色卡 · [角色名]

**视觉参考**：
- @图片[X]：基础外形（默认状态）
- @图片[Y]：[特殊状态1]（如：战斗状态 / 惊奇状态 / 受伤状态）
- @图片[Z]：面部/关键特征特写

**完整锁定描述（建模/跑偏修正用）**：
[从头到脚的完整白描，必须包含以下要素]
- 【头部】[发型 / 发色 / 面部特征 / 五官]
- 【躯干】[材质 / 颜色 / 款式]
- 【四肢】[形态 / 比例]
- 【永久附属物】[武器 / 道具 / 纹身 / 义肢 / 配饰]

**镜头短锚定描述（40-70字，常规生成单元使用）**：
[压缩版，保留最能防换脸/换形的关键识别特征，删除重复材质和非必要否定约束]

**否定约束**：
- 中文禁令：[一条条列出不要生成什么]
- 英文 --no 参数：[用于 MJ/Kling 的负向词列表]

**差异化状态列表**：
| 状态 | 触发场景 | 与基础外形的差异 |
|---|---|---|
| 默认 | [场景类型] | [一致 / 微调描述] |

**视觉冲突提示**（供场景设计参考）：
- 主体色与背景冲突：[需规避的背景色]
- 光线冲突：[需规避的光线类型]
- 尺度冲突：[需规避的场景物体比例]
- 其他：[任何可能造成视觉融合的情况]

**叙事功能定位**（供剧本阶段调用）：
- 戏剧功能：[一句话定位]
- 触发规则：[永远是 X / 永远不是 Y]
- 多角色合奏：[与其他角色的物理关系]
- 叙事禁区：[剧本中不能让角色做的事]

**自然动势倾向**（供分镜阶段调用）：
| 编号 | 动势类型 | 白描指令 |
|---|---|---|
| ST-01 | 基础移动 | [自然白描，禁止学术词] |
| ST-02 | ... | ... |
```

## 2. 场景卡标准模板

### 模板结构

```markdown
### 场景卡 · [场景名]

**视觉参考**：
- @图片[X]：场景全景定场图
- @图片[Y]：场景灯光氛围参考（如有）

**时间段**：[白天 / 清晨 / 黄昏 / 夜晚]

**地点类型**：[室内 / 室外 / 工业 / 自然 / 梦幻自然 / 微观生态（仅用户明确要求时）]

**核心环境元素**（2-3 个最重要的视觉元素）：
1. [元素1：具体描述]
2. [元素2：具体描述]
3. [元素3：具体描述]

**光线来源**：[月光 / 日光 / 荧光灯 / 丁达尔光束 / 冷光 / 暖光]
**光线色温**：[冷蓝 / 暖黄 / 中性白 / 金色]
**光线方向**：[侧逆光 / 顶光 / 正面光 / 逆光]
**地面材质**：[苔藓 / 草地 / 积水 / 青石 / 沙地 / 木地板；自然地表必须写草簇长短不齐、疏密变化、泥土、落叶、小石子、湿润暗部或踩压痕迹]
**背景元素**：[远处有什么 / 墙面有什么 / 天空有什么]
**固定道具**：[场景中反复出现的物体 / 每次镜头都必须保留的细节]

**纵深层次**（前/中/后景参考）：
- 前景：[少量自然遮挡元素，可轻微虚化但不抢戏]
- 中景：[角色活动的舞台留白区]
- 后景：[可读的风景、空间出口、地标、光源或远处结构，不要完全虚化]

**尺度比例暗示**（仅微观世界适用）：
- 巨物参考：[如 "露珠比拳头还大" / "叶脉粗如树干"]
- 环境物体与角色体型比：[如 "蘑菇伞盖是角色身高的 5 倍"]

**正常尺度空场景规则**（默认必填）：
- 场景图用于后续角色合成，不出现角色、动物、人物、拟人角色或额外生物。
- 默认是正常动画电影空间里的近角色观察，不是微缩世界、微距摄影、显微镜视角或苔藓局部素材。
- 可以使用角色高度附近的观察点、少量前景草叶/雨滴/叶片自然遮挡；必须保留中景表演留白和后景可读风景。
- 草地、苔藓和森林地面必须是真实自然材质：草簇长短不齐、疏密变化，混有泥土、落叶、小石子、湿润暗部和局部踩压痕迹；禁止塑料草皮、人工草坪、网格草地或重复编织纹理。
- 禁止默认写“微缩、微距、巨型叶片、露珠湖泊、极浅景深、背景完全虚化、只拍一小块苔藓/叶片/水滴、塑料草皮、网格草地”。

**视觉兼容说明**（主要角色外形与本场景的视觉兼容性）：
- [角色A]：[兼容性描述，如"棕色绒毛与本场景翠绿苔藓形成良好对比"]
- [角色B]：[同上]

**场景生图提示词**：
[按当前工具选择输出一种：即梦中文场景生图提示词 或 MJ英文场景生图提示词。一次只输出一种，不要双语并列。]

**场景氛围词**（用于阶段三风格描述复制）：
[如：清晨柔和漫反射，金色丁达尔光束穿透，SSS 次表面散射叶片，正常尺度3D动画电影空间，自然景深，后景风景可读；仅用户明确要求时加入微观感]
```

| 编号段 | 用途 | 生成阶段 |

| @图片1-9 | 角色图（优先 @图片1、@图片2 为两个主角原图，其余为状态/特写） | 阶段一后期 |

## 4. 固定要素库的使用规则

### 4-1 锚定规则（强制）

- **角色原图优先**：有角色原图时，生成单元只写 `角色名（@图片X）` + 当前动作/状态。
- **短锚定兜底**：无角色原图时使用镜头短锚定描述；首次建立或跑偏修正时使用完整锁定描述。
- **禁止空泛简写**：不能写"人物同上"、"场景同前"；必须引用具体 @图片或写具体短锚定。
- **禁止临场改写识别点**：同一角色/场景的短锚定核心识别点必须稳定，不要每个单元换一种说法。

### 4-2 差异化状态触发规则

- 默认使用**基础外形**
- 当场景触发特殊状态时（如战斗/惊奇/受伤），切换到对应差异化描述
- 差异化只改变**与基础描述不同的部分**，其余完全一致

### 4-3 @图片引用规则

- 每个涉及角色的生成单元必须挂对应角色原图 @图片；有图时不复制角色外形描述
- 同一场景的连续生成单元，推荐使用 `承接首帧@图片X` 锚定
- **禁止用不存在的 @图片编号**（必须先生成图再写提示词）

## 5. 阶段二的四项核心任务

阶段二 Claude 接收到剧本 + 角色参考图后，必须依次完成：

### 任务一：视觉兼容性检查
对比角色参考图与各场景，标注潜在冲突：
- 颜色冲突（主体与背景同色）
- 光线冲突（角色材质与光源反光）
- 尺度冲突（场景物体与角色比例失调）

### 任务二：场景定场图生成
为剧本中每个主要场景生成定场参考图（需补入 @图片库，编号从 10 起）。

场景图提示词规则：

- 当前模式为“即梦中文”时，每个场景卡只输出 `**即梦中文场景生图提示词**`，必须全中文。
- 当前模式为“MJ英文”时，每个场景卡只输出 `**MJ英文场景生图提示词**`，必须英文。
- 一次只输出一种，不要中英双语并列。
- 场景图提示词服务“手动拿去生图”，不是视频分镜提示词。
- 不写运镜、镜头运动、生成时长、首尾帧、提交策略。
- 重点写：正常尺度场景空间、主体留白、前景/中景/后景、关键道具、光线、材质、色彩、风格、画幅比例。
- 场景图默认用于空场景定场，必须明确“无角色/保留主体活动留白/后景风景可读”；不要写角色、小动物、额外生物。
- 每张场景图必须有明确空间任务：入口、过渡、阻碍、转折、开阔收束之一；同一组场景之间要有雾气、冷暖光、视野开阔度、地形结构或地标大小的递进变化。
- 场景图 prompt 要像场景设计说明，而不是风格词堆叠：写清构图引导线、中景空白区、远景地标或空间出口。
- 如出现草地/苔藓/森林地面，必须明确非均匀自然地表：草叶长短不一、疏密变化、泥土落叶小石子、湿润暗部、局部踩压痕迹。
- 禁止默认微缩/微距/显微镜视角/极浅景深/背景完全虚化/局部苔藓水滴图/塑料草皮/人工草坪/网格草地/重复纹理；只有用户明确要求微观世界时才可使用。

### 任务三：生成所有角色卡
- 基于阶段一的角色参考图
- 使用本 KB-04 模板
- 引用已有 @图片编号

### 任务四：生成所有场景卡
- 从剧本提取所有场景
- 使用本 KB-04 模板
- 引用场景图 @图片编号

## 五、下阶段使用说明

- 阶段三的每个生成单元：
