## 0. 两种运镜的本质区别（必读）

### 路径型运镜 ← AI 当作轨迹指令执行

描述镜头的**具体移动路径和终点**，AI 需要精确计算镜头轨迹。

```
例：镜头从左侧以45度弧线推进，同时向上摇，最终停在人物右肩特写位置
例：镜头从远景缓慢推至中全景，人物停稳后固定
```

**特点**：占用大量算力；动作越复杂，路径型运镜越容易导致动作变形。

### 质感型运镜 ← AI 当作风格标签理解

描述镜头的**整体氛围和质感**，不规定具体轨迹，AI 自主解释执行。

```
例：手持摄影，全程晃动，剧烈呼吸感
例：镜头全程环绕推拉，纪录片写实风格
```

**特点**：几乎不占算力；算力全部释放给动作本身。

### 核心使用规则

| 动作复杂度 | 运镜策略 |
|---|---|
| 极复杂（激烈打斗/高难度技巧） | 固定镜头 + 质感型运镜 |
| 中等（走动/追击/对峙） | 路径型 + 质感型叠加 |
| 极简（静止/对话/情绪） | **固定为主，可叠加质感型**（如"固定中景，手持轻微晃动"） |

**黄金组合公式**：

```
路径型运镜 → 只放在 clip 的起点（静止状态）或终点（动作完成后）
质感型运镜 → 贯穿全程，作为整体风格标签叠加
```

## 1. 静态固定类

镜头机位不移动，画面变化完全依赖主体动作或焦点变化。

| 写法 | 适用场景 |
|---|---|
| 固定机位，仰拍倾斜构图 | 权力关系对峙、压迫感营造 |
| 固定地面特写，焦点始终在X上 | 道具特写定格、余韵收尾 |
| 一个固定长镜头，不做任何剪辑 | 生活流场景、文艺片写实段落 |
| 固定中景，手持轻微晃动 | 对话场景、情绪段落黄金组合 |

**实战原句：**

```
固定地面特写，狼牙棒被扔到地上，后景虚化背景是两三个反派尸体，
女孩的脚步经过镜头走远，镜头焦点始终在狼牙棒上，其它内容全部虚化
```

```
固定机位仰拍倾斜构图，前景是反派虚化的身体，
后景是女孩拿着刀对着反派笑的特写
```

## 2. 跟随运镜类

镜头跟随主体运动，保持相对位置稳定。

| 写法 | 说明 |
|---|---|
| 镜头跟随X背影近景 | 最基础跟拍，跟在人物身后，保持背影构图 |
| 运动镜头正面跟拍X中景 | 镜头在主体前方倒退跟拍，拍正面 |
| 镜头跟随X背影，推近晃动 | 跟拍同时向前推近，制造追赶感 |
| 镜头跟随X走向X，她拐向X，到另一个场地 | 跟随式长镜头，经过拐角揭示新场景 |

**实战原句：**

```
机位在电梯内，女孩身后，手持摄影，全程跟拍女孩背影中近景，
电梯开门，女孩走出，镜头一直跟随，女孩出电梯左转
```

```
镜头跟随蝴蝶中近景背影走在走廊，然后右拐，走进下一个走廊
```

```
运动镜头，正面中景跟拍女孩冷静向前走，她不时左右张望
```

## 3. 推拉变焦类

镜头沿景深轴向前推进或向后拉开，改变与主体的距离感。

| 写法 | 说明 |
|---|---|
| 镜头缓慢向前推进 | 情绪渐入、悬念积累 |
| 镜头快速拉近至X | 突然聚焦，强调冲击 |
| 镜头缓缓拉开至大全景 | 情绪收尾、全景收束 |
| 镜头由虚变实，变焦到X | 焦点转移，注意力引导 |
| 镜头微微过X肩部前推，由虚变实 | 过肩推进，揭示前方内容 |

**实战原句：**

```
透过女孩背影，镜头慢慢前推，后景由虚变实至大厅上的蝴蝶logo
```

```
镜头后景逐渐虚化，变焦到前景的桌子上摆放的两条机械臂
```

```
镜头微微过女孩肩部前推，由虚变实，对面8个反派伫立，虎视眈眈
```

```
承接上个镜头，大全景缓缓拉开
```

## 4. 摇镜类

镜头机位不移动，镜头方向旋转改变画面内容。

| 写法 | 说明 |
|---|---|
| 镜头从X开始逐渐上摇至Y | 从低处往高处扫描，逐步揭示 |
| 镜头缓慢上摇至X低头认真的脸 | 人物登场标准摇镜 |
| 镜头从地面缓慢滑行上移 | 渐进式环境介绍 |
| 反向摇镜，从女孩正面摇到右侧面 | 跟随动作的横向摇镜 |

**实战原句：**

```
镜头从特写开始：妹妹的左手机械臂，姐姐用镊子拧着螺丝，
镜头慢慢上摇至妹妹由虚变实的脸
```

```
镜头从他写字的机械手特写开始，缓慢上摇至他低头认真的脸
```

```
镜头起幅从湿漉漉的地面开始，缓慢向前推进，逐渐上移，
画面出现女孩踉跄的双脚，镜头继续缓慢上摇，露出女孩的全身
```

## 5. 环绕运镜类

镜头围绕主体做弧形运动，制造立体空间感和运动冲击力。

| 写法 | 说明 |
|---|---|
| 镜头全程环绕不间断，尽量卡中景 | 格斗场景标配，保持中景跟住动作 |
| 镜头快速环绕到X | 快速转换视角，切换观察方向 |
| 镜头环绕推近至X正脸特写 | 环绕结合推近，人物情绪揭示 |
| 镜头围绕X主体旋转运镜，最后落幅在背影 | 史诗感长镜头结尾 |

**实战原句：**

```
镜头全程环绕推拉，卡中景
手持摄影，全程晃动，剧烈呼吸感，营造纪录片真实感
```

```
镜头环绕推近至女孩正脸特写，她的眼神十分凶狠
```

```
镜头开始围绕女孩主题旋转运镜，最后落幅在女孩的背影停住
```

## 6. 甩镜类

镜头快速横向或纵向甩动，制造速度感和节奏切换。

| 写法 | 说明 |
|---|---|
| 镜头顺势甩向画面左边 | 快速跟随动作方向，制造冲击 |
| 镜头快速甩回给X | 视线快速切换，节奏加速 |
| 镜头耍向一旁，模仿X摔倒 | POV摔倒模拟 |

**实战原句：**

```
7个反派奔涌而来的声音由远及近，镜头顺势甩向画面左边，反派快速跑过来
```

```
随后快速环绕到她脑后，仰拍反派，透过女孩虚化的头部关系，反派慢慢走近
```

## 7. 动作同步绑定类（千舟独创核心语法）

镜头运动与人物动作实时同步，运镜本身成为叙事的一部分。

| 写法 | 说明 |
|---|---|
| 镜头跟随女孩动作画面同步旋转360° | 人物旋转踢，镜头同步旋转 |
| 镜头快速晃动推移移至女孩侧面 | 人物奔跑冲击，镜头同步甩动 |
| 镜头随后快速环绕到女孩头部后面带头部关系 | 人物转身，镜头同步环绕到背后 |

**实战原句：**

```
女孩踩中第一个反派的大腿，借力腾空翻身踢，踢翻反派，
空中旋转360°后落地，镜头跟随女孩动作画面同步旋转360°，
落地后卡近景
```

```
镜头跟随反派背影头部晃动奔跑，透过虚化的头部关系，
看到女孩站在她的对面
```

## 8. POV 主观视角类

镜头完全模拟角色的眼睛，观众进入第一人称视角。

| 写法 | 说明 |
|---|---|
| 一个POV视角长镜头，镜头全程模仿X的眼睛 | 完全主观视角，高度代入感 |
| 镜头向前晃动，模仿X的奔跑节奏 | 呼吸感同步，奔跑代入 |
| 在即将接近X时，X向前冲，镜头甩向一旁，模仿摔倒 | 受击模拟，震撼冲击力 |

**实战原句：**

```
总计一个POV视角长镜头不剪辑，镜头全程模仿女孩的眼睛。
画面里，boss站在走廊的尽头，镜头向前晃动，模仿女孩的奔跑节奏，
速度越来越快，配合女孩的喘息声，在即将接近boss的时候，
boss向前冲两步，直接一个右拳狠狠打到画面上，镜头甩向一旁，模仿女孩摔倒
```

## 9. 叠化转场运镜类

两个场景之间的过渡运镜，制造时空跳跃或梦境感。

| 写法 | 说明 |
|---|---|
| 由首帧@图片X无缝叠化转场至Y | 场景切换带叠化效果 |
| 从闭眼画面匹配过渡到X场景睁眼 | 梦境/现实切换的匹配剪辑 |
| 镜头围绕女孩头部特写向左慢慢旋转，周围环境开始自然改变过渡 | 旋转中改变场景，梦境感 |

**实战原句：**

```
由首帧@图片2这个画面无缝叠化转场至地上一滩雨水，
透过雨水反射，女孩的身影走过来并停下
```

```
镜头开始围绕女孩头部特写向左慢慢旋转，女孩放下扑克牌，
开始惊恐的微微抬头，周围环境开始自然改变过渡到另一个场地
```
