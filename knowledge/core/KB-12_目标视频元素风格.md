# KB-12 · 目标视频元素风格

> **加载阶段**：阶段二（固定要素库）、阶段三（分镜提示词）。
> **用途**：把参考视频的成片观感迁移到当前 IP，而不是机械套用参考视频里的具体元素、剧情或角色。

---

## 1. 核心判断

当前默认风格不是“微观动画”，而是：

**3D半写实梦幻动物动画**

它的关键观感是：

- 3D CGI 成片质感，不是 2D、日漫或平面插画。
- 半写实动物与童话设定结合：真实毛发、湿润眼睛、细腻材质，同时保留软萌比例和轻微夸张。
- 环境以自然森林、花园、草地、蘑菇林、溪边、树根、花丛等为主，可以梦幻，但不要默认是昆虫视角的微观世界。
- 光线以柔和自然光、清晨/上午漫射光、体积雾、丁达尔光束、自然景深、水光高光为主；背景可以适度变软，但远景结构和风景必须可读。
- 叙事观感是“探索世界 → 遇见新景 → 轻微失衡/互动 → 关系升温 → 温暖收束”，不是单一物件追逐。

**角色规则**：默认不新增第三主角。除非用户明确要求，主角队伍只保留用户指定角色，例如香蕉猫和刀盾狗。

**尺度规则**：只有用户明确要求“微观、小人国、昆虫视角、露珠像湖泊、草叶像森林”等表达时，才把环境写成微观尺度。否则按正常动画电影空间处理：镜头可以靠近角色高度，草叶和花朵可以形成前景包围，但不要把一小块苔藓、叶片或水滴当成完整世界。

**元素规则**：发光植物、湿润苔藓、蘑菇、露珠、溪水、蓝绿薄雾、粉紫花丛等只是参考元素池。每条片子按剧情自然选择 2-4 个即可，不要全部硬塞进同一条短片。

---

## 2. 场景语言锚点

### 2-1 入口感

如果剧本需要“进入新空间”的感觉，可以使用自然环境或角色动作来完成：

- 树根洞口、花丛小路、草坡转角、蘑菇拱门、溪流转弯处、废弃小屋门口、飞船/胶囊舱门等。
- 香蕉猫和刀盾狗从入口附近进入画面，观众跟随他们第一次看见环境。
- 画面气质是童话、柔软、好奇，不是硬科幻大片或强动作冒险。

### 2-2 梦幻自然奇观

场景要从普通森林升级为“梦幻自然奇观”，但不强制微观。可从下列元素中按剧情挑选：

- 粉紫色、淡蓝色、乳白色蘑菇形成前景和中景层次。
- 厚实草地、苔藓、花丛或落叶地面，能承接角色脚步和小型失衡。
- 草地和苔藓不能像塑料网或人工草坪：必须有长短不一的草簇、疏密变化、裸露泥土、落叶、小石子、湿润暗部和局部被踩压的痕迹。
- 叶片、花瓣、藤蔓、树根具备透光和湿润质感，但尺寸不必夸张到巨物。
- 透明露珠、叶面水膜、浅溪、水洼或湖面倒影提供湿润高光。
- 背景可保持蓝绿色薄雾和深远纵深，避免单调纯绿。

同一条短片不需要塞满所有元素。优先保证“湿润、通透、柔光、前中后景层次、角色与环境互动”这五个观感。近景细节不能牺牲风景可读性；场景应让观众看见周围空间的美。

### 2-3 水与光

参考画面的高级感来自“水面 + 湿润高光 + 体积光”：

- 可加入浅溪、水洼、湖面倒影、露珠滑落、叶面水膜。
- 水体偏青绿透明，反射角色和植物时要轻微变形。
- 光线以清晨或上午柔光为主，左上侧逆光最稳定。
- 丁达尔光束穿过树冠和薄雾，花粉微粒在光路中漂浮。

## 3. 剧本阶段提醒

剧本可以写蘑菇森林、溪流、花丛、巨叶、露珠、树根洞口等自然元素，但不要把这些写成资产清单。

如果用户要求 90 秒，必须扩展成多段小旅程，不能只靠拉长停顿凑时长。

| 段落 | 功能 | 推荐元素 |
|---|---|---|
| S1 | 入口与出场 | 草地花丛、树根洞口、花径、舱门、溪边等任选其一 |
| S2 | 第一次发现 | 柔光森林、蘑菇群、花丛、低草坡、发光植物等 |
| S3 | 小型互动 | 蘑菇弹性、水珠滑落、叶片回弹、盾牌触水、花粉飘落等 |
| S4 | 空间转变 | 穿过树根、越过浅溪、进入开阔草地、登上小坡等 |
| S5 | 关系升温 | 一个失衡另一个补位、并肩观察、共享发现 |
| S6 | 温暖收束 | 湖面倒影、光束、水雾、花瓣落下、日出光线等 |

### 3-1 90 秒剧本的最低内容量

当用户输入 90 秒左右的目标时长，剧本必须具备：

- 5-7 个场景，而不是 1 个大事件。
- 10-14 个核心物理事件，而不是 4-6 个慢动作事件。
- 至少 4 次空间状态变化，例如：花丛入口 → 蘑菇林 → 浅溪边 → 树根坡道 → 开阔草地/观景点。
- 至少 3 次角色关系变化，例如：跟随 → 同时被吸引 → 一个失衡另一个补位 → 并肩停下。
- 至少 3 种环境物理反馈，例如：叶片回弹、水珠滚动、蘑菇弹起、水雾洒落、水面涟漪、花瓣落下。

90 秒剧本的时长来源必须是“新空间 + 新互动 + 新物理反馈”，不是“长时间凝视 + 缓慢行走 + 静止呼吸”。

## 5. 分镜阶段约束

### 5-0A 短视频节奏反慢动作规则

参考画面的观感不是“把一个动作慢慢拍很久”，而是“每几秒都有新的空间信息、角色关系或物理反馈”。分镜阶段必须避免把微小动作拍成 20 秒慢动作。

**高风险慢镜头信号**：

- 连续使用“缓慢推进、缓慢抬头、完全静止、凝视、升格、时间暂停”。
- 一个生成单元只完成一个极小动作，如推开叶片、脚踩草地、抬头看露珠。
- 同一场景连续 3 个单元都在草地、蘑菇、露珠之间打转。
- 生成时长 6-8 秒，但画面里只有一个动作和一个主体信息。

**修正原则**：

- 每个 5-8 秒生成单元至少包含 2 个视觉变化：角色位移、环境揭示、道具运动、前景遮挡、焦点转移、光影变化任选其二。
- 单个“发现某物”的段落，成片总时长控制在 6-10 秒；除非该物是全片核心奇观，不要超过 12 秒。
- “升格”只用于 0.5-1 秒关键瞬间，不作为整个 3-6 秒镜头的基调。
- “凝视/完全静止”后必须立刻接一个物理反馈：露珠滑动、叶片回弹、草叶弹起、水面起波、刀盾狗撞上或盾牌轻响。
- 如果用户要直接使用生成视频而不是后期精剪，生成时长应接近成片时长，不要默认加 2 秒冗余。

### 5-0 可采纳的构图与镜头语言

参考截图可采纳的是拍法，不是具体物件。分镜阶段应优先吸收以下镜头语言：

| 拍法 | 画面效果 | 提示词写法方向 |
|---|---|---|
| 角色高度入场 | 角色更贴近自然环境，草叶和花丛有包围感，但仍能看见后方风景 | 镜头接近角色高度，中低角度平拍或微仰拍，保留远景空间 |
| 前景遮挡构图 | 叶片、蘑菇、草叶制造窥视感和纵深 | 前景少量叶片/草叶自然虚化遮挡画面一侧，中景角色清晰，后景薄雾和地标仍可辨认 |
| 大全景奇观定场 | 让观众先看见空间规模，再看见角色位置 | 大全景缓慢拉开，角色位于画面下方或中下部，环境占据大部分画面 |
| 中景双角色关系 | 强调香蕉猫与刀盾狗的队伍关系 | 两个角色同向行走或同向凝视，左右/前后站位明确，相距半身到一身 |
| 近景反应切换 | 用眼睛、鼻尖、盾牌、水珠承接情绪 | 固定近景或中近景，自然景深，背景柔和但空间仍可读，焦点在表情或道具 |
| 水面/露珠反射 | 增加高级感和诗意转场 | 透过水洼/露珠/溪水倒影看到角色或光束，倒影轻微变形 |
| 叶片/蘑菇擦镜转场 | 自然遮挡完成切镜，不靠硬切 | 镜头跟随角色穿过叶片或蘑菇前景，前景短暂遮满画面后揭示新空间 |

### 5-1 镜头比例

60 秒短片推荐镜头构成：

- 25%-35%：大全景/远景，展示梦幻森林、花丛、溪流或空间纵深。
- 35%-45%：中景，展示香蕉猫和刀盾狗一起穿行、停顿、互动。
- 15%-25%：近景/特写，展示眼睛、鼻尖、盾牌、水珠、蘑菇弹性。
- 5%-10%：转场镜头，如黑场、穿过叶片、掠过水面、蘑菇前景遮挡。

### 5-2 必备画面气质

分镜提示词末尾推荐加入：

```
3D半写实梦幻动物动画，真实毛发质感，柔和自然光，梦幻森林，电影级CG质感，角色软萌但不过度卡通，材质细腻，自然景深，背景空间可读
```

光影氛围推荐加入：

```
清晨柔和侧逆光从左上方穿过森林冠层，丁达尔光束与花粉微粒可见，叶片具备次表面散射，水面和露珠产生细小高光
```

### 5-3 负向边界

- 不要默认新增绿色外星小生物。
- 不要把香蕉猫和刀盾狗替换成其他角色。
- 不要把森林做成普通写实树林，必须有柔光薄雾、奇观植物、水光反射、前景遮挡中的至少两项。
- 不要默认微观尺度；只有剧本或用户明确要求时，才按用户原文补充特殊尺度词。
- 不要机械逃生舱、飞船、硬科幻、枪械、爆炸、强对抗；除非剧本明确需要。
- 不要一镜到底只拍脚下露珠，也不要让场景图只剩一小块苔藓/叶片/水滴；必须有可读风景、空间出口和双角色关系镜头。
- 不要把背景全部虚化；治愈感来自“近处角色亲密 + 周围风景美丽”，不是只把焦点钉在角色或一个局部物件上。
- 不要把草地生成成塑料草皮、人工草坪、网格草地或重复编织纹理；自然地表必须有不规则草簇、泥土、落叶、小石子和湿润层次。
- 不要把浓雾、厚雾、冷青绿浓雾、暗水区、深色水面、昏暗断崖、黑暗峡谷当作治愈童话的默认场景机制；雾只能是极轻透明水汽或远景空气透视。
- 治愈童话场景优先采用“童话地标物 + 柔软自然地表舞台 + 前景叶片/花草包围 + 可读森林纵深 + 少量透明空气感 + 暖光斑”的构成。每场至少有一个一眼可记住的地标物，例如向日葵、雏菊、粉色蘑菇、树根拱门、浅溪石头、藤蔓秋千、苔藓木桥。
- 不要连续多个镜头都用固定中景；必须穿插大全景、低机位跟拍、前景遮挡、近景反应和物理细节特写。

## 6. 一句话风格锁定

**香蕉猫和刀盾狗在正常尺度的半写实梦幻自然环境中展开软萌轻冒险，画面靠近角色但保留可读风景；场景由童话地标物、柔软自然地表舞台、前景花草包围、可读森林纵深、极轻透明空气感和暖光斑构成；保持真实毛发、细腻材质、柔和体积光、湿润水光、自然景深和高质量 3D 梦幻动画电影质感；不默认微观、浓雾、暗水或阴冷峡谷，除非用户明确要求。**
