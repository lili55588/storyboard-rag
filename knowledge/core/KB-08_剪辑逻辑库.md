## 1. 叙事连接词系统

跨片段衔接的核心语法，控制镜头间的时间和空间连续性。

| 连接词 | 含义 | 使用场景 |
|---|---|---|
| `承接首帧@图片X` | 本片段起点画面与参考图一致 | 跨片段动作衔接 |
| `承接上个镜头` | 时间线连续，紧接上一个画面 | 同场景连续剪辑 |
| `延续首帧@图片X` | 场景状态延续，非精确动作衔接 | 场景延续，动作可有差异 |
| `延续@图片X` | 同上，更宽松 | 同场景再次出现 |
| `由首帧@图片X无缝叠化转场至` | 叠化转场的起点锚定 | 时空跳跃、回忆 |
| `对切首帧@图片X的反打镜头` | 反打，注明轴线参考 | 对话场景正反打切换 |
| `尾帧@图片X` | 本片段结束画面锚定 | 控制片段出点 |

详细用法见 KB-05 第 5 节。

## 2. 三种合并/拆分策略（V6 §四核心）

> **V6 重要修正**：将 V5 的"打包提交"拆分为两个独立概念——「无缝合并」和「分镜打包」。

### 2-1 核心概念区分

| 概念 | 含义 | 使用格式 | 目的 |
|---|---|---|---|
| **无缝合并** | 景别和运镜完全不变，多个动作写在同一段描述里，无剪辑点 | 直接用"随后""紧接着"串联 | 让 AI 生成连贯动作 |
| **分镜打包** | 景别或运镜有变化，一次提交包含多个镜头，让即梦自动生成剪辑节点 | 用"镜头一（Xs）""镜头二（Xs）"格式 | 减少提交次数，节省配额 |
| **独立提交** | 动作激烈或复杂，算力不够分，必须拆开分别提交 | 各自单独提交 | 保证动作质量 |

### 2-2 三种情况对号入座

**情况一：景别 + 运镜都不变 → 无缝合并**

同一个镜头位置，同一个运镜风格，只是动作有先后顺序：

```
固定中景，手持轻微晃动。
（主体/站位/场景描述...）
女孩蹲下来检查伤口，随后抬起头扫视四周，紧接着缓缓起身，手按刀柄保持警戒。
→ 一次提交，生成时长按总成片时长 +2 秒填写
```

**情况二：景别或运镜有变化，但动作不激烈 → 分镜打包**

用"镜头一/二"格式，一次提交，让即梦在切换点自动生成剪辑：

```
总计两个镜头。
镜头一（6秒，成片时长）：固定中景，手持轻微晃动。
  女孩在审讯室里缓缓走近，绕到反派椅子后方，双手搭在椅背上低头俯视。
镜头二（5秒，成片时长）：承接上个镜头，仰拍大特写，
  切近女孩面部，她冷冷开口，嘴唇轻动说话。
→ 成片总时长 11 秒，生成时长填 13 秒（11+2，受平台上限取整）
→ 一次提交
```

> **分镜打包的时长计算**：各镜头成片时长之和 ≤ 11 秒，加 2 秒冗余后 ≤ 13 秒。

**情况三：景别或运镜变化，且含激烈动作 → 必须独立提交**

激烈打斗、高速追逐等复杂动作需要大量算力：

```
激烈打斗核心段 → 单独提交，生成时长按规则 A 填写
定场/收尾特写  → 单独提交，生成时长按规则 A 填写
→ 后期剪辑拼接
```

### 2-3 极短镜头（成片 ≤ 2 秒）的处理

极短镜头永远不单独提交，并入最近的相邻镜头：

```
❌ 单独提交一个成片 1.5 秒的闪切镜头 → 浪费生成配额
✅ 把该 1.5 秒闪切并入相邻镜头的分镜打包或无缝合并里 → 一次提交，后期截取
```

## 3. 正打/反打系统

对话和对峙场景的核心剪辑语法。

### 3-1 基础正打反打

```
镜头一（正打）：过A肩拍B，A肩虚化在画面左下角，B实焦居右
镜头二（反打）：过B肩拍A，B肩虚化在画面右下角，A实焦居左
注意不要越轴
```

### 3-2 关系反打变体

```
透过A的头部关系仰拍B脸部特写      → 仰拍反打（强调B的优势）
透过A关系俯拍B头部特写            → 俯拍反打（强调B的弱势）
透过A虚化的肩膀俯拍半蹲的B        → 俯拍侧身反打
前景是A虚化的头部，后景是B        → 景深反打
```

### 3-3 实战原句

```
承接首帧@图片4 的反打镜头（注意不要越轴），
透过铁锤关系仰拍蝴蝶脸部大特写，注意蝴蝶左侧嘴角有血渍
```

```
承接上个镜头（注意轴线关系），反派身旁一坑水的特写，
透过水面映射出女孩的身影
```

## 4. 轴线关系与 180 度法则（V6 §六）

### 4-1 什么是轴线

两个角色之间存在一条虚拟的"轴线"（对话轴/运动轴）。在同一场景的所有镜头里，摄影机必须保持在轴线**同一侧**拍摄，否则两人的左右关系会发生翻转，观众感知到空间错乱。

### 4-2 越轴的典型错误

```
镜头A：男性在画面左侧，女性在画面右侧，男性看向右方（看着女性）
镜头B（越轴）：男性在画面右侧，女性在画面左侧，男性却还在看向右方
→ 观众感知：两人突然"穿越"到对方位置，空间错乱
```

### 4-3 如何在提示词里控制轴线

**方法一：每个镜头明确写各人物在画面左右的位置**

```
镜头一：男性居画面左侧，女性居画面右侧，男性看向右（看向女性方向）
镜头二（反打）：女性居画面左侧，男性居画面右侧，
注意：此处是反打，男性在右，女性在左，轴线关系不变
```

**方法二：使用过肩反打时明确说明**

```
承接上个镜头，过男性右肩反打女性（注意不要越轴），
男性右肩虚化在画面左下角，女性实焦居画面右侧中央
```

**方法三：在提示词中直接写"注意不要越轴"**

在多人对话或打斗的连续镜头中，可以直接在结尾加一句：

```
注意保持轴线关系，不要越轴
```

## 5. 信息分层揭示策略

用连续镜头把场景信息分层喂给观众，制造悬念和节奏感。

### 5-1 从局部到整体

```
第一层：地上的面具（结果/线索）
第二层：女孩的脚踩进来（主角出现）
第三层：上摇揭示两旁尸体（场景规模）
第四层：手上的蝴蝶刀（凶器/能力）
第五层：变焦远处奄奄一息的反派（目标/任务）
```

### 5-2 从静到动

```
固定场景/物品特写  →  人物入画  →  动作开始
（建立环境）          （主角登场）    （叙事启动）
```

### 5-3 从结果到原因（倒叙式揭示）

```
先看到满地尸体  →  再看到女孩经过  →  观众自行推断战斗过程
```

## 6. 格斗动作拆解语法

激烈打斗场景的逐秒时间轴写法，精确控制动作节奏。

### 6-1 时间轴格式

```
1-3秒：【攻击方】+【攻击动作】，【防守方】+【防御/反应】
3-6秒：【攻击方】+【攻击动作】，【防守方】+【防御/反应】
```

### 6-2 动作元素四要素

```
攻击方向（正面/左侧/右侧/背后）
+ 攻击方式（横扫/下砸/直刺/旋踢）
+ 防御动作（闪避/挡住/借力）
+ 结果（打倒/弹飞/撞墙）
```

### 6-3 武术流派标注（增加动作可信度）

```
泰拳：膝撞、扫踢、肘击
以色列格斗术：咽喉刺、快速制压
空手道：手刀、掌底击
咏春：组合连拳、超速出手
柔术：十字固、关节反锁、背摔
```

### 6-4 实战原句

```
1-3秒：反派一正面挥棒下砸，女孩滑步闪避 + 左钩肝 + 右上勾将其打倒
3-6秒：反派二从左后方横扫，女孩摇闪 + 直拳 + 摆拳
6-10秒：反派三挥空重心前移；女孩后撤步闪避 + 直拳反击 + 钩拳终结
10-15秒：反派四从右侧猛砸，女孩右直拳打中面门，追击左右直拳连击（2拳），最后右摆拳击倒
```

## 7. 情绪节拍控制

控制整个视频的情绪节律。

| 情绪状态 | 镜头策略 | 节拍感 |
|---|---|---|
| 情绪积累/悬念 | 长镜头不剪辑，缓慢推进 | 压抑、期待 |
| 情绪爆发/打斗高潮 | 快速环绕+推拉+手持晃动 | 冲击、混乱 |
| 情绪收尾/反思 | 固定镜头，升格，静止 | 沉淀、余韵 |
| 情绪反差（回忆插入） | 文艺片风格固定镜头，温暖光线 | 对比、忧伤 |
| 压迫感建立 | 仰拍+倾斜+前景遮挡 | 窒息、无力 |

### 交叉剪辑蒙太奇设计

```
现实线（打斗复仇）：手持晃动 + 冷蓝色调 + 只有音效
回忆线（温情瞬间）：固定镜头 + 暖光 + 文艺片气质
→ 两条线交叉剪辑，制造情感张力
```

## 8. 专属特写插入时机（动感核心）

在关键动作后切入局部特写，制造景别跳跃即动感。

### 8-1 标准插入时机

```
重击命中后  →  切受击方面部大特写（痛苦反应）
武器亮出时  →  切武器特写（道具强调）
关键转折时  →  切主角眼神大特写（心理变化）
场景切换前  →  切环境细节特写（叙事过渡）
```

### 8-2 专属特写生成单元是什么

在关键动作发生后，专门切入一个生成单元，只拍该动作最具冲击力的局部细节。这个单元本身不推进剧情，但它**创造了景别跳跃**，这个跳跃本身就是动感来源。

### 8-3 标准动感拆分模板

```
生成单元1（成片4秒，生成6秒）· 定场起手  → 路径型推镜（静止承接）
生成单元2（成片4秒，生成6秒）· 交手核心  → 固定 + 质感晃动（算力全给动作）
生成单元3（成片3秒，生成5秒）· 冲击特写  → 仰拍/倾斜 + 升格（景别跳跃）
```

## 9. 转场描述（V6 §九）

### 9-1 硬切（默认，最常用）

不需要特别描述，即梦默认在镜头切换处硬切。分镜打包时直接写：

```
镜头一（6秒）：...
镜头二（5秒）：承接上个镜头，...
```

### 9-2 叠化转场（Dissolve）

```
由首帧@图片1这个画面无缝叠化转场至地面一滩雨水的特写
```

适用于时间跳跃、回忆闪回、场景转换。

### 9-3 甩镜转场

```
镜头快速向右甩出，画面模糊的瞬间切入新场景
```

### 9-4 转场描述的位置

写在前一个镜头结束、后一个镜头开始的衔接处：

```
镜头一（5秒）：…女孩转身出画。
→ 叠化转场
镜头二（6秒）：叠化进入夜晚场景，女孩站在建筑门口……
```

## 10. 水面/镜面反射叙事用法

利用物理反射制造诗意叙事距离。

```
水面反射  →  用于过渡段落，制造时间流逝感/诗意
镜子倒影  →  双重影像，内省/内心分裂/关系映照
雨水积坑  →  配合打斗，力量真实感（被打到水溅起）
玻璃折射  →  窥视感，隔阂感
```
