# KB-07 · 镜头语言库

> **加载阶段**：阶段三（分镜提示词）。
> **来源**：旧「运镜与镜头语言知识库.md」模块二 + V6 手册第一/第二章（站位与虚实）。
> **用途**：提供景别、构图、虚实、入画出画、权力关系等镜头语言的词表与写法。

---

## 1. 景别词表（从大到小）

| 景别名称 | 拍摄范围 | 核心用途 |
|---|---|---|
| 大全景 | 整个环境，人物极小 | 定场、环境交代 |
| 全景 | 人物全身+周围环境 | 人物出场、动作全貌 |
| 中全景 | 人物全身，腿部可见 | 打斗全貌、双人关系 |
| 中景 | 腰部以上 | 对话、常规叙事 |
| 中近景 | 胸部以上 | 情绪+动作兼顾 |
| 近景 | 肩部以上 | 情绪表达、重要对话 |
| 特写 | 头部+肩部 | 表情捕捉、情绪强调 |
| 大特写 | 整张脸 | 情绪爆发、关键反应 |
| 超大特写 | 面部局部（双眼/嘴角） | 极致情绪、细节强调 |

### 景别与视角的叠加写法

```
仰拍大特写        → 强势/压迫感（被拍者有力量）
俯拍大特写        → 弱势/被控制感
倾斜构图大特写    → 紧张/不稳定/危机
平拍中景          → 客观叙事/平等关系
```

## 2. 站位描述体系（V6 §一）

> 参考图只锁定角色外形，不锁定空间位置。**不管有没有参考图，每个多人镜头都必须写站位。**

### 2-1 站位描述的四个维度

**① 横轴位置（左中右）**：用画面三等分来定位每个人物

```
女孩站在画面左侧 1/3，boss 站在画面右侧
两人各占画面一侧，中间留有空隙
```

**② 纵深层次（前景 / 中景 / 后景）**：明确谁在前、谁在后

```
前景是虚化的反派背影，后景实焦是女孩
前景左侧是虚化的门框，中景是面向镜头的男主
```

**③ 相对距离**：

```
两人相距约一臂距离，正面对峙
boss 站在女孩正前方两步处，微微俯视
```

**④ 主次关系（谁是实焦主体）**：

```
镜头焦点在女孩身上，boss 身影虚化在后景
```

### 2-2 站位描述的写作位置

站位描述紧跟在主体外形之后，动作之前，**独立成一句**：

```
【运镜】固定中景，手持轻微晃动。
【主体】30岁亚洲男性，寸头，黑色练功服，白腰带。
        25岁亚洲女性，高马尾，红色短打功夫上衣，黑色武术裤。
【站位】两人相距两米正面对峙，男性居画面左侧，女性居右侧，镜头焦点对准两人中间。
【动作】男性右腿猛地抬起踢出旋踢，女性急速左闪……
```

### 2-3 过肩镜头的站位写法（最易翻车）

```
镜头过男性右肩拍女性，
男性右肩虚化占画面左下角 1/5，
后景实焦是女性面部大特写，居画面右侧偏中，
两人相距约一米。
```

### 2-4 单人镜头也要写站位

```
女孩居画面中央偏左，背景是虚化的走廊，镜头焦点在她的面部。
```

## 3. 构图语法库

### 3-1 倾斜构图

画面有意倾斜一定角度，打破水平稳定感。

**适用场景**：对峙、拷问、紧张情绪、心理失衡

**实战写法**：
```
倾斜构图，过倒地的反派关系反打女孩
承接上个镜头，切近女孩大特写的脸，倾斜构图
仰拍，倾斜构图 15 度，升格，手持轻微晃动
```

### 3-2 过肩反打构图

前景人物肩部虚化，后景实焦对话方。标准对话剪辑构图。

**写法公式**：
```
过 X（左/右）肩反打 Y，
X 的（左/右）肩虚化占画面（左/右）下角约 1/5，
后景实焦是 Y 的（景别），居画面（位置）
```

**实战原句**：
```
过铁锤头部关系俯拍他的左臂大特写，铁锤看着自己的左手机械臂外壳已经损坏
过女孩肩部反打 boss，boss 一只在低头写字不理会，女孩逐渐走近
镜头过女孩的肩部反打她的手部特写
```

### 3-3 关系镜头构图（核心语法）

用前景人物/物体作框架，建立前景与后景的叙事关系。

**写法公式**：
```
透过 X（虚化/实焦）的 Y 关系，（仰拍/俯拍/平拍）Z（景别）
```

**实战原句**：
```
透过虚化的背景，能看到铁轨两旁横七竖八的躺着反派尸体
透过女孩虚化的头部关系，反派出现，从远处快速跑来偷袭
透过铁锤关系仰拍蝴蝶脸部大特写，蝴蝶向下俯视着铁锤
透过将军背影关系仰拍反打女孩大特写脸部，倾斜构图
透过爬在地上的女孩关系反打 boss 中近景
```

**权力关系编码规律**：
```
透过弱势角色仰拍强势角色  → 强化强势一方的压迫感
透过强势角色俯拍弱势角色  → 强化弱势一方的无助感
透过物体关系拍人物         → 人物与物体建立叙事联系
```

### 3-4 画面三等分定位

明确主体在画面中的横向位置。

```
居画面左侧 1/3      → 视线引向右侧（期待）
居画面右侧         → 视线引向左侧（回望）
居画面中央         → 主体感强，权威/孤独
画面只有半边脸     → 神秘/张力/局限视野
前景遮挡左 1/3      → 偷窥感/被遮蔽的威胁
```

**实战原句**：
```
构图只有半张脸在镜头左侧，背景是虚化的走廊
前景神秘女孩背影入画，遮挡住镜头左 1/3，递给女孩一个白手帕
```

## 4. 仰拍/俯拍与权力关系

| 拍摄角度 | 被拍者感受 | 典型使用场景 |
|---|---|---|
| 强烈仰拍 | 高大/强势/压迫 | boss 登场、女孩获胜时 |
| 微仰拍 | 略有优势 | 对峙场景 |
| 平拍 | 平等/客观 | 对话、叙事过渡 |
| 微俯拍 | 略弱势 | 受伤/被审问 |
| 强俯拍 | 弱势/被控制/渺小 | 女孩被打倒时、趴地 |

**实战原句**：
```
低角度仰拍反打，面部超大特写，她的眼神从无助到愤怒（开场倒地仰拍）
反打超大特写仰拍，倾斜构图（boss 对比弱势方时的仰拍）
俯拍 boss 手里的蝴蝶刀大特写（boss 俯视女孩时）
```

## 5. 虚实关系与景深三层次（V6 §二）

> Seedance 理解"虚实"的前提是你**明确写出来**。没有写虚化的元素，模型默认全部实焦，景深感消失。

### 5-1 景深三层次公式

```
前景（虚化/实焦）+ 中景（主体，通常实焦）+ 后景（虚化/实焦）
```

**完整示例**：
```
前景是虚化的反派蜷缩的身体，
中景是实焦的女孩侧面近景，镜头焦点在她的面部，
后景是虚化的废旧铁轨和杂草。
```

### 5-2 焦点转移（变焦）的写法

```
镜头焦点从前景虚化的机械臂逐渐后拉变实，
同时后景的女孩面部从虚化逐渐变为实焦大特写。
```

### 5-3 虚实的常见用途

| 用途 | 写法 |
|---|---|
| 强调主体 | 前后景虚化，主体实焦 |
| 制造层次感 | 前景实焦道具/手，后景虚化人物 |
| 转移注意力 | 焦点从 A 转移到 B（从虚变实） |
| 过肩/过人关系 | 前景过肩人物虚化，后景对话人物实焦 |

### 5-4 焦点转移写法

```
前景由实变虚，变焦到后景 X 的特写
后景由虚变实，镜头焦点转到 X
镜头焦点始终在 X 上，其它内容全部虚化
```

**实战原句**：
```
前景是趴在地上的反派不断挪动虚化的身体，后景是左手拿着匕首不断接近的女孩
前景是虚化的机械臂，后景是女孩看着扑克牌的脸
镜头后景逐渐虚化，变焦到前景桌子上摆放的两条机械臂
```

## 6. 入画/出画控制

控制主体进出画面的时机，隐含剪辑节奏信息。

| 类型 | 写法 | 作用 |
|---|---|---|
| 自然入画 | X 自然入画，顺势… | 人物从镜头边缘进入，自然感 |
| 强制遮挡入画 | X 背影前景入画，遮挡住镜头左 1/3 | 突然遮挡，制造悬念 |
| 出画 | X 左侧/右侧出画 | 标记剪辑点，离开场景 |
| 跑出画面 | 女孩冲跑出画 | 追逐感，节奏加速 |

**实战原句**：
```
女孩的右脚自然入画，顺势踩中面具
前景神秘女孩背影入画，遮挡住镜头左 1/3
女孩慢慢起身，双手插兜，左侧出画
```

## 7. 环境动态细节写法

用环境物理现象增加画面真实感和情绪渲染。

| 环境元素 | 写法 | 情绪作用 |
|---|---|---|
| 风 | 冷风不断拂动她的头发 | 孤独、冷峻 |
| 雨水 | 打斗时地上坑洼的雨水不断被溅的到处飞 | 写实、力量感 |
| 水面反射 | 透过雨水反射，女孩的身影走过来 | 诗意、叙事距离感 |
| 镜子倒影 | 透过姐妹的头部关系拍卧室镜子，镜子里映出姐妹俩 | 内省、文艺感 |
| 秋千晃动 | 只剩两只秋千在微晃动 | 失落、余韵 |
| 汗珠/血迹 | 额头不断渗出汗珠，嘴角渗出血渍 | 真实、生理感 |
| 脚步震动水面 | 水面被女孩的脚步震出一丝波纹 | 静谧中的力量 |

## 8. 升格镜头（慢动作，V6 §八）

### 8-1 升格的写法

直接在运镜/视角部分写明：
```
升格镜头，固定特写，手持轻微晃动
```

或在动作描述中写明：
```
女孩起身，镜头切入升格，她将刀干练收起，头发随动作轻轻飘起
```

### 8-2 升格的适用场景

| 场景 | 是否适合升格 |
|---|---|
| 关键打击瞬间（拳击命中/刀刺入） | ✅ 强烈推荐 |
| 情绪特写（眨眼/微笑/泪水） | ✅ 推荐 |
| 走廊行走、过渡镜头 | ❌ 不推荐 |
| 激烈连续打斗段 | ❌ 不推荐 |
| 收尾定格（主角转身离开） | ✅ 适合 |

**升格不能替代场景描述**——其他所有描述（站位、构图、虚实、动作）仍然必须写完整。
