# KB-11 · 终极分镜导演执行手册

> 加载阶段：阶段三（shot / 分镜提示词）。  
> 目标：替代 KB-05 至 KB-10 的执行型规则。只保留 How，不解释 Why。

---

## -1. 执行优先级

当规则冲突时，按以下顺序执行：

1. 用户本次明确要求。
2. 平台硬限制：纯中文、字数、生成时长、16:9。
3. 动作衔接：首尾帧、上一单元终点、下一单元起点。
4. 一致性：角色原图 @图片、场景图 @图片、必要短锚定。
5. 运镜、景别、构图、越轴。
6. 风格、质感、音效。

禁止为了补全格式而编造不存在的 @图片编号、角色、场景、道具、镜头结果。缺少图片时，使用 `【待补图@图片X】` 占位；缺少动作或资产信息时，保持描述保守，不自行扩写新设定。

---

## 0. 最高铁律

- 输出纯中文提示词；运镜/视角必须放在每条完整提示词第一句。
- 单条完整提示词控制在 120-240 中文字符。超出时删重复材质、形容词、否定约束、已由参考图承担的信息。
- 每个生成单元必须写：情绪功能、成片时长、生成时长、提交策略、首帧/尾帧计划、完整提示词、给下一个单元的接口。
- 生成时长 = 成片时长 + 2 秒冗余；最高写 13 秒。成片超过 11 秒时，生成时长直接写 13 秒。
- 第一帧必须有人，第一秒必须有动作。独立空镜禁止单独提交，必须并入有人镜头。
- 成片 2 秒以内的极短镜头并入相邻单元，不单独提交。
- 收尾镜头成片时长优先控制在 3 秒以内。
- 不写“同上”。一致性交给角色原图 @图片、场景图 @图片、首尾帧和稳定站位。
- 不机械粘贴资产卡。完整锁定描述只用于首次建立、无参考图或跑偏修正。

---

## 1. 完整提示词公式

固定顺序：

`运镜/视角 → 主体图片引用 → 站位与构图 → 动作时间轴 → 场景短锚定 → 必要光影/环境动态 → 风格质感 → 精简质量约束 → 音效指令 + 生成时长 + 16:9`

执行规则：

- 【运镜/视角】一句内完成，放最开头。可含景别、机位、升格、手持晃动。
- 【主体图片引用】有角色原图时只写 `角色名（@图片X）`，不要复制外形描述；没有角色图或跑偏修正时才使用短锚定/完整锁定描述。
- 【站位与构图】多人镜头必须写横轴位置、前中后景、虚实关系、相对距离。
- 【动作时间轴】按起点 → 过程 → 终点写，不只写静态结果。
- 【场景短锚定】写地点、光线方向、关键道具、空间尺度即可。
- 【光影/环境动态】只写会影响动作或情绪的环境变化。
- 【风格质感】选 1 组风格词，不堆多组。
- 【质量约束】常规写“面部五官稳定，比例自然，动作连贯不僵硬”。跑偏时再加否定约束。
- 【结尾参数】必须以音频方案 + 生成时长 + 16:9 收束。

## 2. 一致性与锚定

### 2-1 角色/场景一致性

- 有角色原图：只写 `角色名（@图片X）+ 当前动作/状态`，不复制角色外形描述。
- 无角色原图：使用角色短锚定；首次建模或跑偏修正时使用完整锁定描述。
- 有场景参考图：`@图片X + 场景短锚定 + 光线方向`。
- 无场景参考图：使用完整场景锁定描述。
- 同一角色的 @图片编号必须稳定；同一场景的光线方向和关键空间识别点必须稳定。
- 文字描述必须与 @图片一致。不要挂站立图却写躺倒，不要挂白天图却写夜晚。

### 2-2 首尾帧语法

- 起点承接上一单元：`承接首帧@图片X`。
- 终点需要精确控制：`尾帧@图片X`。
- 起点和终点都要锁：`承接首帧@图片X，尾帧@图片Y`。
- 首帧只锁起点，不能替代动作、站位、运镜和焦点描述。
- 下一单元起点必须等于上一单元终点，不跳步。

### 2-3 反打与越轴

- 同一场景连续多人镜头，角色 A/B 左右站位必须保持一致。
- 正反打写法：`承接首帧@图片X 的反打镜头（注意不要越轴），过A肩拍B`。
- 过肩镜头必须写：前景谁虚化、后景谁实焦、后景人物在画面左/中/右。
- 连续对话、对峙、打斗镜头必须主动写“注意不要越轴”。

## 3. 拆分、合并与时长

- If 景别和运镜都不变：无缝合并。用“随后 / 紧接着”串联动作，不写镜头一镜头二。
- If 景别或运镜变化，但动作不激烈：分镜打包。写“镜头一（X秒，成片时长）/ 镜头二（X秒，成片时长）”。总成片时长 ≤ 11 秒，生成时长 = 总时长 + 2 秒。
- If 景别或运镜变化，且动作激烈：独立提交。每个复杂动作单独成生成单元。
- If 只有试探性动作、不改变关系或结果：删除或并入主动作。
- If 关键动作后需要动感：插入专属特写生成单元，成片 2-4 秒，使用景别跳跃。

---

## 4. 运镜与镜头选择字典

### 4-1 运镜选择

- If 激烈打斗 / 高难度动作 / 快速坠落 / 复杂群体运动：固定机位 + 质感型手持晃动。禁止路径型推拉跟拍。
- If 对话 / 对峙 / 情绪反应：固定中景或固定近景 + 手持轻微晃动。
- If 简单行走 / 探索 / 慢速靠近：路径型推镜、跟拍、缓慢上摇可用。
- If 定场 / 世界建立：大全景或低角度路径型推镜；第一秒仍必须有人或角色动作。
- If 收束 / 余韵：固定远景或缓慢后拉，动作减少，环境动势保留。
- 路径型运镜只放在 clip 起点或动作完成后；复杂动作过程中不要同时安排路径型运镜。

### 4-2 景别选择

- 建立空间：大全景 / 全景。
- 角色全身动作：中全景。
- 对话或双人关系：中景 / 过肩中景。
- 情绪变化：近景 / 特写。
- 冲击瞬间或关键道具：大特写 / 超大特写。
- 童话自然环境：接近角色高度的中低角度、少量前景自然虚化更优先；远景森林、水道、树根、光束等空间结构仍要可读。只有用户或剧本明确微观尺度时，才写成昆虫视角或巨物尺度反差。

### 4-3 构图与虚实

- 多人镜头固定格式：`A在画面左侧中景，B在画面右侧后方约X身位，前景X虚化，焦点在Y`。
- 关系镜头格式：`透过X的虚化前景，拍Y的近景/特写`。
- 权力压迫 / 巨物感：仰拍 + 倾斜构图。
- 脆弱 / 俯视观察：俯拍。
- 空间深度：前景轻微虚化 + 中景实焦 + 后景退远但结构可读。
- 焦点转移：先写起点焦点，再写“焦点从X缓慢转到Y”。

---

## 5. 动作与节奏

- 动作必须按时间顺序写：起始姿态 → 触发 → 身体反应 → 结果姿态。
- 微表情/微动作只写 1-2 个关键点。不要堆呼吸、眼神、手指、衣角、光斑。
- If 动作依赖道具：写清道具位置、接触点、受力结果。
- If 环境驱动剧情：写清环境变化如何改变角色动作。
- If 情绪变化：用停顿、后退、靠近、视线锁定、身体僵住来表达，不写内心。
- If 关键冲击 / 水花炸裂 / 瞬间失衡 / 情绪峰值：可用专属特写或升格。

---

## 6. 升格、转场与音频

### 6-1 升格

- 只在关键打击命中、道具碰撞、坠落接触、情绪峰值特写、重要水花/碎裂/爆发瞬间使用。
- 常规走路、普通转头、试探动作禁止升格。
- 升格写在运镜位置：`升格镜头，固定特写，手持轻微晃动`。
- 升格不能替代站位、动作、场景和焦点描述。

### 6-2 转场

- 默认硬切，不必写。
- 叠化写法：`无缝叠化转场至……`，用于梦境、回忆、时间流逝。
- 甩镜写法：`镜头跟随X快速甩向Y，形成甩镜转场`，用于快速空间切换。
- 转场描述放在镜头开头或两个镜头之间。

### 6-3 音频

- 写实动作 / 紧张段：`只需要音效，无配乐`。
- 文艺情绪段：`只需要背景配乐，无音效`。
- 高潮段：`音效与配乐同时存在，配乐低混，音效突出`。
- 后期自行配音：`无音效无配乐，完全静音`。
- 音频句必须放在最后，并紧接生成时长和比例。

## 7. 风格与质感字典

默认 3D 半写实梦幻动物动画：

- `3D半写实梦幻动物动画，真实毛发质感，柔和自然光，梦幻森林，电影级CG质感，角色软萌但不过度卡通，材质细腻，自然景深，背景空间可读`
- `正常尺度动画电影空间，接近角色高度的中低角度，前景叶片或蘑菇少量自然虚化，柔和自然光，湿润空气感，前中后景层次清楚，远景风景可读`
- `松弛但不拖沓的节奏，动作轻微迟钝，环境细节安静漂浮`

微观尺度（仅用户或剧本明确要求时使用，默认禁用）：

- 不提供默认可复制模板；只有用户或剧本明确要求特殊微观尺度时，才按原文补充尺度反差。

动作/紧张：

- `写实动作风格，高对比冷色调，手持摄影呼吸感，冲击瞬间清晰`
- `固定机位，手持强烈晃动，尘土/水花/碎屑强化动感`

文艺/余韵：

- `柔和漫射光，低饱和色彩，自然景深，背景空间可读，安静留白`
- `侧逆光轮廓，空气微粒可见，情绪克制`

质量约束：

- 常规：`面部五官清晰稳定不变形，比例自然，动作连贯不僵硬`
- 跑偏修正：加入角色卡否定约束，但只加与本次错误相关的 2-4 条。

## 8. 最终自检

- 提示词是否 120-240 字？
- 运镜是否第一句，音频 + 生成时长 + 16:9 是否最后一句？
- 有角色原图时是否只写角色名（@图片X），而不是复制角色外形？
- 多人镜头是否写了站位、虚实、相对距离？
- 连续多人镜头是否保持左右关系，并写“注意不要越轴”？
- 动作是否从起点写到终点？
- 生成时长是否等于成片时长 + 2 秒，且不超过 13 秒？
- 景别/运镜变化是否已判断无缝合并、分镜打包或独立提交？
- 第一帧是否有人，第一秒是否有动作？
- 是否删除了不改变结果的试探性动作和独立空镜？
