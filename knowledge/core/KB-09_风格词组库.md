## 5. 质量约束词组（人物镜头必加，V6 §11 第十位）

```
面部五官清晰稳定不变形，眼睛嘴型不扭曲，
人体结构正常，比例自然，动作连贯不僵硬
```

**使用规则**：
- 所有包含角色的生成单元必须在提示词第十位加上
- 即使是空镜/纯环境镜头，也推荐加上"画面结构正常，比例自然"
- 这段是 Seedance 2.0 官方推荐的质量锚点，不要省略

## 6. 音效与配乐指令（V6 §十）

### 6-1 四种音频方案

| 方案 | 写法 | 适用场景 |
|---|---|---|
| 只有音效，无配乐 | `只需要音效，无配乐` | 打斗、紧张对话、写实纪录风格 |
| 只有配乐，无音效 | `只需要背景配乐，无音效` | 文艺片、情绪段落 |
| 音效 + 配乐 | `音效与配乐同时存在，配乐低混，音效突出` | 动作场面的戏剧性增强 |
| 完全静音 | `无音效无配乐，完全静音` | 用于后期自行添加音轨 |

### 6-2 音效描述的写法

可以描述具体的音效内容，而不只是说"有音效"：

```
脚步声由远及近，刀刃划过金属的摩擦声，玻璃破碎声
```

```
雨水溅落的声音，沉闷的击打声，反派倒地的声音
```

### 6-3 梦幻动物动画的推荐音效

```
草叶轻擦的窸窣声，露珠落地的清脆声，
微风拂过的轻柔声，小动物脚步的细微声响
```

```
蘑菇弹性的回弹声，水面被脚步震动的涟漪声，
苔藓被踩压的轻微摩擦声
```

### 6-4 有音频参考时的对话写法

挂上音频参考（`@音频X`）后可以直接写台词：

```
女孩冷冷说：我再问你最后一遍，谁指使你们来的。
```

**没有 `@音频` 时，不写台词文字，只写嘴部动作**：

```
女孩开口说话，嘴唇随说话节奏轻轻动着，眉头微皱，语气激烈
```

## 11. 风格词的组装位置（V6 §11 第七/九位）

在完整写作公式中，风格词分为两个位置：

- **第八位：光影氛围** → 从 §8、§9 挑选
- **第九位：风格调性** → 从 §1-§4 挑选
- **第十位：质量约束** → §5 必加
- **最后：音效指令** → 从 §6 挑选

**组装示例**：
```
...（前面的运镜/主体/站位/动作/场景）...
清晨柔和漫反射光，金色丁达尔光束穿透森林冠层（第八位：光影氛围）。
3D半写实梦幻动物动画，真实毛发质感，柔和自然光，电影级CG质感（第九位：风格调性）。
面部五官清晰稳定不变形，人体结构正常，比例自然，动作连贯不僵硬（第十位：质量约束）。
只需要音效，无配乐（音效指令），6 秒（生成时长），16:9
```
