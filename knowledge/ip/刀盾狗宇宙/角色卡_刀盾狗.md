## 一、完整锁定描述（建模/跑偏修正用）

小巧的双足奇幻动物角色。角色尺度由剧情场景决定，不默认微观。

- 【头顶/脸部】一张幼犬的脸（类似博美犬），直接长在毛茸茸的圆躯干上，**没有可见的脖子**；黑色湿润的小鼻头；一对深色圆眼睛；表情天真忠诚
- 【躯干】一个圆滚滚的椭球体，**全身覆盖密集的棕色犬类绒毛**（dense coarse brown dog fur）
- 【四肢】四肢极其粗短，双足直立行走（bipedal stance）
- 【永久附属物】右手握着一把小铁短剑（small iron sword），左手拿着一面圆木盾，盾面带金属包边与铆钉（round wooden shield with metal rim and studs）
- 【无尾】没有可见的狗尾巴
- 【核心神态】天真、忠诚、略显呆萌

## 一-补、镜头短锚定描述（常规生成单元使用）

双足直立的小巧刀盾狗，幼犬脸直接长在棕色圆滚绒毛躯干上，无颈无尾，四肢粗短，右手小铁剑，左手圆木盾带金属包边，神情天真慢半拍。

## 二、否定约束（跑偏修正时使用，常规镜头不全文复制）

**中文禁令**：
- 禁止四足爬行姿态（必须双足直立）
- 禁止把狗毛错误渲染为刺猬的尖刺
- 禁止鼓起腮帮子（保持犬科吻部自然生理结构）
- 禁止人类牙齿
- 禁止丢弃右手的剑或左手的盾（永久持有）
- 禁止展现精湛剑术或主动劈砍攻击
- 禁止四肢着地爬行、扒住树皮、手脚并用（双手被武器永久占用）
- 禁止精准格挡等武术动作（只能是本能闭眼举盾）
- 禁止伸手撑地（摔倒时只能靠腰腹翻身）
- 禁止毛发因惯性抖动等 AI 会自动解算的冗余物理反应

**英文 --no 参数**：
```
--no hedgehog spikes, sharp quills, quadrupedal, walking on four legs, dropping weapons, human hands, human teeth, puffed cheeks, martial arts, precise blocking, aggressive attacking, anthropomorphic exaggerated expressions, cartoonish morphing
```

## 四、动作动势白描词库（storyboard 阶段按场景挑选复制）

> 【最高警告】：描写动作时必须使用自然、呆萌的白描语言！绝对禁止使用"几何体""次级形变"等学术词！绝对禁止将狗毛描写为刺猬尖刺！

| 动作类型 | 白描原句 |
|---|---|
| 基础移动 | 靠两只小短腿高频倒腾着小碎步跑动 |
| 急停 | 圆滚滚的身体控制不住地往前一倾 |
| 本能防御 | 缩起脖子，把左手的圆木盾举过头顶或挡在身前，双眼紧闭，充满惊慌和呆萌感 |
| 被动弹起 | 没有任何主动起跳能力；被外力颠飞时像毛茸茸实心球一样在空中手足无措 |
| 受击翻滚 | 在地上像毛球一样笨拙地往后翻滚或滑行 |
| 笨拙爬起 | 靠圆滚滚腰腹力量翻身，双手依然死死握着武器，不伸手撑地 |
| 疑惑反应 | 呆呆歪着脑袋，耳朵跟着抖动两下 |
| 惊吓反应 | 本能地紧闭双眼，缩紧身体 |
| 默认神态 | 忠诚无辜地望着前方 |

## 五、视觉冲突提示（阶段二场景设计参考）

| 冲突类型 | 规避方案 |
|---|---|
| 大面积棕色/土色背景 | 棕色绒毛身体会与背景融合 → 场景禁用纯棕/土黄主色调，需要高对比环境 |
| 金属反光背景 | 可能抢走剑与盾的金属视觉焦点 → 优先使用柔和自然光 |
| 等比例场景物体 | 不再默认禁止；按剧本决定尺度，普通动画森林也可以使用正常比例的石头、花草、木桥等 |
| 平光渲染 | 会让圆滚滚毛球失去立体感 → 优先使用侧逆光勾出毛发轮廓 |

## 六、叙事功能定位（阶段一剧本参考）

- **戏剧功能**：忠诚但迟钝的笨拙护卫，队伍里的可靠跟班，永远慢半拍
- **叙事触发规则**：永远是跟从者，绝对不会主动接触或招惹未知元素
- **行动动机**：跟随本能 + 保护同伴的迟钝意识
- **危险反应**：第一反应是闭眼防守（举盾缩头），而不是主动出击
- **物理站位**：永远处于香蕉猫侧后方约 1-2 个身位的跟随安全区

**绝对禁止项（叙事层）**：
- 禁止主动走在队伍最前面探路
- 禁止展现武林高手般的精湛剑术或主动劈砍攻击
- 禁止丢弃手中的剑与盾
- 禁止任何语言；允许少量普通身体交流，如举盾靠近、停步回头、把盾挪到同伴脚边，但不要写成人类式夸张手势

**多角色合奏动态（与香蕉猫）**：
- 香蕉猫惹出麻烦 → 刀盾狗被迫遭遇冲撞、水溅或惊吓
- 香蕉猫走在 C 位 → 刀盾狗始终处于侧后方跟随区
- 场景物件飞来 → 刀盾狗本能举盾承接，香蕉猫呆立观望

## 七、一致性质检清单（每个生成单元完成后自查）

1. □ 角色保持双足直立行走，没有四条腿走路？
2. □ 全身覆盖密集的棕色狗毛，没有被错误渲染为刺猬尖刺？
3. □ 幼犬五官自然，没有鼓起腮帮子或露出人类牙齿？
4. □ 右手一直握着剑，左手一直拿着圆木盾，没有丢弃？
5. □ 动作描述是自然白描（如"高频倒腾小碎步"），没有学术词汇？
6. □ 遇险时只是本能闭眼举盾，没有主动挥剑或精准格挡？
7. □ 摔倒时像圆球笨拙翻滚，没有伸手撑地？
8. □ 走路体现了小短腿步频极快的特征？
9. □ 多角色场景中，始终跟在香蕉猫的侧后方？
10. □ 没有描写"毛发因惯性抖动"等冗余物理反馈？
