# 香蕉猫宇宙 · 世界观设定

> **加载阶段**：阶段二（视觉开发）。
> **用途**：提供默认成片气质与环境边界；不作为剧情铁律。

---

## 1. 世界基调

一句话：**3D半写实梦幻动物动画**。

这是一个没有对白、以好奇心、环境变化和轻微物理互动推动的童话动画世界。角色可以在森林、花园、草地、蘑菇林、溪边、树根小路、花丛空地等自然环境里探索。画面应有真实毛发、细腻材质、柔和自然光、湿润水光、浅景深和电影级 3D CG 质感。

默认观感更接近“童话微缩花园”：角色像小生命在花朵、叶片、蘑菇、草坡、溪边石头和小道具之间穿行，环境物常常明显大过角色，形成“小生命面对大世界”的包围感和玩法。

但这不是显微镜素材、微距局部摄影或只拍一小块苔藓。画面必须保留可读中景舞台、前景自然遮挡和清楚的后景空间。

核心美学参考：皮克斯式高质量 3D 动画 + 半写实自然材质 + 梦幻童话森林。

## 2. 物理法则

### 2-1 尺度规则

- 角色整体小巧软萌，但真实尺寸不固定，由剧本和画面场景决定。
- 普通森林、花园、草地要以角色高度附近观察，花朵、叶片、蘑菇、草坡、溪边石头或小道具可明显大过角色，形成童话微缩比例。
- 童话微缩不等于显微镜视角；不要把场景退化成一片苔藓、一滴水珠或完全虚化背景。

### 2-2 环境物理属性

环境通过可见物理反馈推动剧情：

| 环境属性 | 物理效果 | 叙事功能 |
|---|---|---|
| 湿滑/低摩擦（青苔石、浅滩、带露水的叶） | 试探着挪步 → 重心失控 | 制造小危机 |
| 高弹性（蘑菇、柔软草皮） | 触碰 → 回弹 → 二次震颤 | 制造轻喜剧 |
| 遮挡/柔软（叶片、花丛、垂枝） | 探头/绕行/短暂消失 | 制造空间发现 |
| 刺激型（花粉、水滴） | 喷嚏、闭眼、缩脖 | 物理笑点 |
| 水光反射（水洼、溪流、露珠） | 倒影、涟漪、光斑变化 | 诗意转场 |

### 2-3 重力与动势

- 角色没有主动武术能力，动作偏笨拙、迟钝、自然。
- 失衡、摔倒、被弹起都应像真实小身体受力，而不是夸张卡通形变。
- 刀盾狗的剑盾是永久附属物，可以本能举盾保护同伴，但不要写成武术高手。

## 3. 时空设定

### 3-1 时间段

允许：清晨漫射光 / 上午柔和日光 / 黄昏金色时刻 / 夜晚月光 + 萤火虫。

避免：无来源平光、工业强光、过度赛博或硬科幻灯光，除非用户明确要求。

### 3-2 地理类型

允许：

- 梦幻森林地表：草地、苔藓、蘑菇、蕨类、花丛。
- 水边：浅溪、水洼、湖边、青苔石、湿润木桥。
- 植物架构：树根、藤蔓、叶片、花瓣、蘑菇平台。
- 自然遮蔽所：蘑菇伞下、树洞、翻倒陶罐、枝叶缝隙。

避免：

- 大面积纯黄色/纯棕色背景，会让香蕉猫或刀盾狗主体融合消失。
- 脏乱废土、恐怖血腥、强对抗捕食场面。
- 现代都市和复杂人类社会空间，除非用户明确要求。

### 3-3 光影规则

- 优先使用侧逆光勾勒角色轮廓。
- 可使用丁达尔光束、次表面散射、花粉微粒、湿润高光。
- 暗部不能沉闷，需被植物绿色、粉紫花丛或水面反光柔化。

## 4. 叙事驱动力

核心驱动：环境变换 + 纯粹好奇心 + 同伴关系。

真正推动叙事的元素：

- 环境变换：从花丛走到水边，从树根坡道走到开阔草地。
- 物理连锁：A 的笨拙触碰 → 环境反馈 → B 被动响应。
- 关系变化：跟随 → 一起发现 → 一个失衡另一个补位 → 并肩停下。

避免：

- 打怪升级、暴力血腥、重反派冲突。
- 对白、旁白、内心独白。
- 把 90 秒写成单一物件追逐或长时间凝视。

---

## 5. 调性禁区

- 不要让香蕉猫和刀盾狗说话。
- 不要用字幕解释它们的内心。
- 不要写成人类式复杂手势；可用停步回头、靠近半步、举盾、并肩看向同一处等普通身体动作。
- 不要把刀盾狗写成精湛剑客；它主要是本能保护和笨拙补位。
- 不要把香蕉猫写成聪明指挥者；它主要由好奇心触发行动。
